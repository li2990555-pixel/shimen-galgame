## 1. 产品概述

### 1.1 一句话定位

> **把「专注学习」变成异步竞技——和好友/全服玩家进行多轮次的学习远征，比谁的策略更优、谁更专注。**

### 1.2 核心价值

| 用户痛点 | 解决方案 |
|---------|---------|
| 一个人学不下去，缺乏监督和竞争 | **异步 Ghost 竞技**——你知道有人和你同时在学 |
| 番茄钟枯燥，难以坚持 | **远征制 + 策略决策**——每轮备战有游戏化反馈 |
| 学习成果不可见 | **多维度数据 + 段位 + 赛季** |
| 想卷但缺少氛围 | **好友战书 + 全服匹配 + 排行榜** |

### 1.3 差异化定位

```
                    专注对战（异步）
                        │
                        │
       LearnClash       │      卷王对决（我们）
       (Quiz异步)        │      (专注时长异步)
                        │
       答题对战 ─────────┼──────── 专注时长对战
                        │
                        │
       Focus Town       │      Forest
       (自习在线)        │      (个人专注)
                        │
                    专注自习（同步）
```

**关键差异**：
- vs Focus Town：我们是对战而非自习，是异步而非同步
- vs LearnClash：我们是专注时长而非答题，是远征制而非单局
- vs Forest：我们是社交竞技而非个人工具

### 1.4 参考游戏映射

| 游戏 | 借鉴点 |
|------|--------|
| **The Bazaar** | Ghost 异步竞技、经济系统、信息不对称、Build 策略 |
| **格斗游戏（拳皇/街霸）** | 选人→放狠话→3-2-1→KO 结算的仪式感节奏 |
| **Slay the Spire** | 多轮次决策、事件牌、装备构建 |
| **Clash Royale** | 奖杯系统、赛季重置、晋级赛 |
| **Marvel Snap** | Snap 下注机制、短局制、最终揭晓悬念 |

## 2. 目标用户

### 2.1 用户画像

**核心用户：备考学生（考研/考公/留学）**
- 年龄 18-28 岁
- 每天需要 4-8 小时专注学习
- 有固定的学习时间段
- 使用过 Forest/Focus Town 等专注工具
- 需要外部压力和竞争驱动

**扩展用户：职场学习者**
- 年龄 22-35 岁
- 下班后自学技能/考证
- 时间不固定，需要灵活的学习工具
- 轻度游戏化即可驱动

### 2.2 典型场景

| 场景 | 用户 | 使用模式 |
|------|------|---------|
| 每晚 8-10 点和好友一起卷 | 备考学生 | 好友战书 |
| 周末大段时间深度学习 | 备考学生 | 4-5 轮远征 |
| 午休碎片时间学一会儿 | 职场人 | 1-2 轮远征 |
| 想验证自己的学习效率 | 所有用户 | 天梯匹配 |
| 一个人但想学 | 所有用户 | 影子挑战 |

## 3. 核心玩法

### 3.1 核心循环

```
┌─────────────────────────────────────────────────────────┐
│                    学习远征（Campaign）                    │
│                                                          │
│  一场远征 = N 轮（每轮 = 一个番茄钟）                      │
│                                                          │
│  Round 1:  备战(2min) → 专注(25min) → 复盘(1min)        │
│  Round 2:  备战(2min) → 专注(25min) → 复盘(1min)        │
│  Round 3:  备战(2min) → 专注(25min) → 复盘(1min)        │
│  Round 4:  备战(2min) → 专注(25min) → 复盘(1min)        │
│                                                          │
│  ─────────────────── 终局结算 ───────────────────         │
│  全面数据对比 + KO 动画 + 段位变动 + 奖励                 │
└─────────────────────────────────────────────────────────┘
```

### 3.2 异步匹配机制

```
用户完成一场远征后，系统匹配另一个完成远征的用户的 Ghost 数据
进行对比：

  你的远征 4 轮数据 ──┐
                      ├── 系统对比 ──→ 胜负判定
  对手 Ghost 4 轮数据 ─┘

匹配规则：
  • 段位相近（±200 ELO）
  • 轮次相同（3/4/5 轮匹配对应轮次）
  • 秒匹配（无需等待，Ghost 已存在）
```

### 3.3 核心竞争维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 总专注时长 | 40% | 所有轮次专注时间总和 |
| 中断次数 | 25% | 中断越少越好 |
| 任务完成率 | 20% | 任务完成比例 |
| 最高专注倍率 | 15% | 最长持续专注的倍率系数 |

## 4. 游戏流程详解

### 4.1 第一步：远征开始

```
用户操作（约 30s）：

1. 选择英雄风格（3 选 1）
   ⚡ 闪电型 — 25min 模式效率 +10%（适合短时爆发）
   🛡️ 壁垒型 — 中断损失 -30%（适合稳定输出）
   🔥 火焰型 — 连胜奖励 +50%（适合状态型选手）

2. 选择远征长度
   3 轮（75min 专注）  /  4 轮（100min）  /  5 轮（125min）

3. 选择赌注
   10 币（娱乐） / 50 币（挑战） / 200 币（生死）
```

### 4.2 第二步：每轮备战（2min）

备战的四个标签页，用户可以自由切换：

#### 4.2.1 事件牌

```
从事件池中随机抽一张：

📗 正面事件（30%）
  "安静环境"     → 专注效率 +15%
  "学习歌单"     → 专注效率 +10%
  "状态良好"     → 起始精力 +20%

📕 负面事件（30%）
  "手机震动"     → 中断概率 +25%
  "睡眠不足"     → 精力恢复 -30%
  "邻座聊天"     → 专注时长 -3min

📘 抉择事件（40%）
  "咖啡还是茶？"      → 选效率 or 精力
  "休息还是冲刺？"    → 选保守 or 高风险
  "难题还是基础？"    → 选高难度高回报 or 稳过
```

#### 4.2.2 商店

```
每轮商店刷出 4 件物品，可购买：

基础物品（固定池）：
  ⏱ 番茄钟延长器    30币  专注时长+5min
  🛡️ 专注护盾       25币  抵挡一次中断扣分
  🎯 目标锁定       35币  专注效率+25%
  ⚡ 双倍卡         50币  专注币收入×2
  💊 精力药剂       15币  精力消耗-30%
  📋 任务加速       20币  任务奖励+50%
  🔮 预言水晶       40币  预览对手装备
  🏆 连胜护符       60币  赢得+20段位分

刷新机制（每轮限 2 次）：
  • 花费 5 专注币刷新全部 4 件商品
  • 和 The Bazaar 的刷新机制一致
```

#### 4.2.3 装备配置

```
装备栏：3 格
背包：从商店购买的物品存放于此

操作：
  • 从背包点击物品 → 装备到空槽位
  • 点击已装备物品 → 卸下回背包
  • 每轮开始前可自由调整

策略示例：
  • 前期：经济装备（双倍卡+任务加速）
  • 后期：爆发装备（延长器+目标锁定）
  • 防御型：护盾+精力药剂
```

#### 4.2.4 任务设定

```
可选任务（每轮选 1-4 个）：

  📖 高数复习        常规 +10 币
  📝 英语单词 50 个   常规 +10 币
  📄 阅读论文 2 篇    困难 +20 币
  🧮 刷题 50 道      困难 +20 币

策略点：选 1 个困难任务 vs 选 2 个常规任务
  困难完成率高但失败损失大
  常规稳定但收益低
```

### 4.3 第三步：专注（25min）

```
┌──────────────────────────────┐
│          25:00               │  ← JetBrains Mono 大字
│      第 3 轮 · 专注中         │
│    ████████░░░░░░░░          │
│                              │
│  [暂停]           [结束]      │
│                              │
│  📋 任务 ▸（默认折叠）        │
└──────────────────────────────┘

战时规则（严格）：
  ✅  大字计时器
  ✅  当前轮次标识
  ✅  暂停/结束按钮
  ✅  任务清单（默认折叠）
  ❌  无对手信息
  ❌  无游戏动画
  ❌  无弹窗通知
  ❌  无段位/排名/社交
```

**专注倍率机制**（战中不显示，复盘揭晓）：

| 持续专注时间 | 倍率 |
|-------------|------|
| 0-10min | ×1.0 |
| 10-20min | ×1.2 |
| 20-30min | ×1.5 |
| 30min+ | ×2.0 |

### 4.4 第四步：复盘（1min）

```
本轮展示内容：

1. 我的本轮数据
   • 专注时长
   • 中断次数
   • 任务完成数
   • 实际专注倍率
   • 本轮回合收入

2. 对手有限信息
   • 完成轮次
   • 累计专注时长
   • 累计中断次数（不展示本轮单独数据）
   • 对手段位

3. 累计比分
   • 双方累计专注时长的进度条对比
   • 领先/落后标识

信息控制原则：
  让用户知道"大概的局势"，但不清楚对手的装备/策略/具体数据
  保持悬念到终局结算
```

### 4.5 第五步：终局结算（高潮）

```
结算揭晓流程（每步间隔 0.5-1s）：

Step 1: 总专注时长揭晓
  "你的总专注: XXmin vs 对手: XXmin"

Step 2: 逐轮数据对比
  R1: 25 vs 22
  R2: 25 vs 20
  R3: 25 vs 24
  R4: 25 vs 22

Step 3: 关键数据
  中断次数 / 任务完成率 / 最高倍率

Step 4: KO 判定 + 奖励
  🏆 你赢了！/ 😤 惜败
  +XX 段位分 / -XX 段位分
  +XX 专注币
  解锁成就/物品
```

### 4.6 完整时间预算

```
4 轮远征总计：

  Round 1 备战    2min    ← 游戏时间
  Round 1 专注    25min   ← 学习时间
  Round 1 复盘    1min    ← 轻度游戏
  Round 2 备战    2min
  Round 2 专注    25min
  Round 2 复盘    1min
  Round 3 备战    2min
  Round 3 专注    25min
  Round 3 复盘    1min
  Round 4 备战    2min
  Round 4 专注    25min（决战轮）
  Round 4 复盘    1min
  终局结算        2min    ← 游戏高潮
  ─────────────────────
  总计：~114min
  纯学习：100min（88%）
  游戏化：14min（12%）
  
  游戏化时间全部落在「休息时间」
  不侵占学习时间
```

## 5. 策略系统

### 5.1 策略维度总览

```
每轮备战的决策空间：

维度              决策点                    策略深度
────────────────────────────────────────────────
英雄选择         风格 × 远征长度 × 赌注       初始策略（一次定全局）
事件应对         如何利用/降低事件影响        风险应变（每轮不同）
商店消费         买什么 + 刷不刷新            经济博弈（核心深度）
装备配置         3格 × 8种物品组合            Build 构建（排列组合）
任务选择         常规/困难 × 数量             风险管理
对手信息         有限数据的解读推理            信息博弈
```

### 5.2 经济系统

```
专注币的流动：

收入来源：
  完成番茄钟基础收入    +1 币/min
  任务完成奖励          +10~20 币/个
  对战胜利              +赌注×2
  连胜奖励              +连胜场次×5
  每日签到              +20 币
  赛季结算              +段位×100 币

支出：
  商店购物              15-60 币/件
  商店刷新              5 币/次（每轮限2次）
  对战下注              10/50/200 币

限制：
  不可充值购买
  每日专注币上限 200 币
  专注币余额 = 努力证明
```

### 5.3 策略成长曲线

```
新手期（前 5 局）：
  • 随便选选，体验流程
  • 下最小的注
  • 胜负看运气

进阶期（5-20 局）：
  • 开始理解装备 synergy
  • 根据事件调整策略
  • 开始关注对手数据

资深期（20+ 局）：
  • 精确的商店经济管理（刷新 vs 存钱）
  • 针对对手策略做反制
  • 赛季策略（季初猛冲 vs 季末保段）
```

## 6. 对战模式

### 6.1 好友战书

```
流程：
  用户完成远征 → 点击「下战书」
  → 选择好友
  → 用本次远征数据生成 Ghost
  → 好友收到推送：「小明向你发起挑战！」
  → 好友接受 → 72h 内完成远征
  → 数据对比 → 结算

设计要点：
  • 下战书本身是一次"晒成绩"
  • 好友的 Ghost 成为靶子
  • 社交裂变自然发生
```

### 6.2 天梯匹配

```
流程：
  点击「排位赛」→ 系统匹配段位相近的 Ghost
  → 用户远征 → 自动对比 → 结算

匹配规则：
  • ELO 匹配（±200）
  • 轮次匹配（同轮次）
  • 秒匹配（Ghost 已存在）
  • 匹配到的对手匿名显示

段位系统：
  青铜(0-599) → 白银(600-999) → 黄金(1000-1299) →
  铂金(1300-1599) → 钻石(1600-1899) → 大师(1900-2199) → 王者(2200+)
  
  ELO 参数：
    K-factor: 前 10 场 40，之后 20
    初始段位分: 600（青铜III）
```

### 6.3 影子挑战（单人）

```
流程：
  点击「影子挑战」→ 系统用用户上周同时间专注数据生成 Ghost
  → 用户远征 → 和过去的自己对比
  → 赢了：证明在进步

设计要点：
  • 无需好友、无需网络匹配
  • 自我超越的长期留存核心
  • 适合新手、社交焦虑用户
```

### 6.4 BOSS 战（双人合作 — V3）

```
流程：
  和好友组队 → 选 BOSS（2h/5h/10h 共同目标）
  → 各自完成远征 → 两人专注时长之和达标 → BOSS 击败
  → 获得合作限定徽章

BOSS 设计：
  恶龙(2h) → 海妖(5h) → 外星人(10h)
  
设计要点：
  • 合作而非竞争
  • 适合好友一起卷
  • 让不习惯竞争的用户也能参与
```

## 7. 游戏化系统

### 7.1 段位系统

| 段位 | 分区间 | 小阶 | 特点 |
|------|--------|------|------|
| 🥉 青铜 | 0-599 | 3 | 初始段位 |
| 🥈 白银 | 600-999 | 3 | 大部分用户 |
| 🥇 黄金 | 1000-1299 | 3 | 认真卷的用户 |
| 💎 铂金 | 1300-1599 | 3 | 前 20% |
| 🏅 钻石 | 1600-1899 | 3 | 前 5% |
| 👑 大师 | 1900-2199 | 2 | 前 1% |
| 🏆 王者 | 2200+ | 1 | 前 0.1% |

**晋级赛**：升大段位需打晋级赛（3局2胜）
**保级**：长期不活跃缓慢扣分，专注可积累护盾

### 7.2 成就系统

| 成就 | 条件 | 奖励 |
|------|------|------|
| 初战告捷 | 完成第一场远征 | 头像框 |
| 完美远征 | 全程无中断获胜 | 100 币 |
| 马拉松选手 | 单场 5 轮远征 | 限定皮肤 |
| 早起卷王 | 早 6-8 点完成 10 场 | 称号 |
| 最佳拍档 | 和同一好友对战 50 场 | 专属称号 |
| 百战卷王 | 累计 100 场 | 限定立绘 |
| 王者降临 | 赛季结算王者 | 赛季限定皮肤 |

### 7.3 赛季系统

```
周期：每月 1 天 → 月末

赛季主题：
  🥋 武道会赛季 / 🚀 星辰赛季 / 🏯 忍者赛季 / 🧙 魔法赛季

赛季内容：
  • 赛季限定任务
  • 赛季排行榜（好友榜 + 全区榜）
  • 赛季限定皮肤（过期绝版）
  • 赛季结算奖励（基于最终段位）

段位重置：
  王者 → 铂金I / 大师 → 黄金I / 钻石 → 白银I / 铂金以下不变
```

### 7.4 英雄之路

```
Lv.1（0h）   初始形象
Lv.5（10h）  新发型
Lv.10（50h） 新服装 + 入场特效
Lv.20（200h）专属称号
Lv.30（500h）传说皮肤
Lv.50（1000h）定制KO特效
```

### 7.5 反作弊与防沉迷

```
反作弊：
  • 专注中随机弹出"还在吗？"确认
  • 加速度传感器检测手机状态
  • 同一账号单设备登录
  • 新账号前 10 场不计入排位

防沉迷：
  • 每日对战上限（免费 10 场，付费 20 场）
  • 连续专注 2h 后强制 15min 休息
  • 0:00-6:00 仅开放影子挑战
```

    ✗ 游戏化元素/动画

### 11.3 Ghost 匹配逻辑

```
用户完成远征 → 系统保存为 Ghost 记录
↓
其他用户发起匹配 → 系统查找段位相近的 Ghost
↓
匹配成功 → 用户开始远征
↓
用户完成后 → 比较两者数据
↓
胜负判定 → 更新段位分 → Ghost 失效

特点：
  • Ghost 是只读数据，不会被修改
  • 一个 Ghost 可被多次匹配（但仅第一次的匹配者使用后失效）
  • Ghost 有效期 7 天（之后需要重新生成）
```
