# 卷王对决 — 完整游戏流程与策略系统设计

> 核心思路：每个"对局"不是一个番茄钟，而是一整局游戏
> 参考系：The Bazaar（异步自走棋）+ Slay the Spire（卡牌肉鸽）

---

## 一、核心结构：一局游戏 = 一场「学习远征」

### 基本概念

```
一局「学习远征」(Campaign)
  ├── Round 1: 备战 → 专注 → 复盘
  ├── Round 2: 备战 → 专注 → 复盘
  ├── Round 3: 备战 → 专注 → 复盘
  ├── Round 4: 备战 → 专注 → 复盘（决战轮）
  └── 终局结算：双方远征数据全面对比
```

每一局游戏你和对手都同时进行一场"学习远征"，各有 4 轮（4 个番茄钟）。**你们不是实时进行的，而是异步的。** 你先完成你的 4 轮，系统去匹配另一个也完成了 4 轮的玩家来对比。

### 和 The Bazaar 的对应

```
The Bazaar              卷王对决
─────────────────        ─────────────────
一局游戏 (Run)           一场「学习远征」
Day 1-10                 Round 1-4
PvE 打怪                 专注执行
PvP 和其他玩家 Ghost      和对手的 Ghost 对比
Shop (买装备)             商店（买学习道具）
经济管理                   专注币管理
「吃鸡」                  赢了对局
```

## 三、Round 内三阶段详解

### 阶段 1：备战（2-3 分钟）

这是策略深度所在。每轮备战包含以下环节：

#### 1a. 事件牌（随机，15s）

```
你翻开一张事件牌，影响本回合：

📗 正面事件        📕 负面事件          📘 抉择事件
────────────        ──────────          ──────────
"安静环境"         "手机震动"           "咖啡还是茶？"
专注效率 +15%      中断概率 +20%        选 +20%效率或 +15%精力

"学习歌单"         "邻座聊天"           "推一把还是稳一手？"
专注效率 +10%      专注时长 -5min       选 +25%收益或 -50%中断风险

"状态良好"         "睡眠不足"           "攻克难题还是刷基础？"
起始精力 +20%      精力恢复 -30%        选高难高回报或低难稳过
```

**设计要点**：
- 事件牌让每回合都不一样（新鲜感）
- 正面/负面平衡，避免过于随机
- 抉择事件让玩家有策略选择
- 事件仅影响本轮，决策压力小但有趣

#### 1b. 商店（30s-60s）

```
┌──────────────────────────────┐
│   🛒 专注商店 — Round 2       │
│                                │
│  你的专注币: 85                │
│                                │
│  ┌──────────────────────────┐ │
│  │ ⏱ 番茄钟延长器   30币     │ │
│  │   本回合专注时长 +5min    │ │
│  ├──────────────────────────┤ │
│  │ 🛡️ 专注护盾       25币    │ │
│  │   抵挡一次中断（免扣分）   │ │
│  ├──────────────────────────┤ │
│  │ 📋 任务加速       20币    │ │
│  │   任务完成奖励 +50%      │ │
│  ├──────────────────────────┤ │
│  │ 💊 精力药剂       15币    │ │
│  │   本回合精力消耗 -30%    │ │
│  └──────────────────────────┘ │
│                                │
│  [刷新商店: 5币]  [跳过]       │
└──────────────────────────────┘
```

**物品设计**：

| 物品 | 效果 | 价格 | 类型 | 参考来源 |
|------|------|------|------|---------|
| 🍅 番茄钟延长器 | 本回合专注时长 +5min | 30 | 消耗品 | The Bazaar 武器 |
| 🛡️ 专注护盾 | 抵挡一次中断扣分 | 25 | 消耗品 | 护甲装备 |
| 📋 任务加速 | 任务完成奖励 +50% | 20 | 消耗品 | 经济增益 |
| 💊 精力药剂 | 精力消耗 -30% | 15 | 消耗品 | 药水 |
| 🔮 预言水晶 | 预览对手当前局装备 | 40 | 消耗品 | 侦察手段 |
| 🎯 目标锁定 | 本回合专注效率 +25% | 35 | 消耗品 | 攻击增益 |
| ⚡ 双倍卡 | 本回合专注币收入 ×2 | 50 | 消耗品 | 经济卡 |
| 🏆 连胜护符 | 赢得本局额外 +20 段位分 | 60 | 被动 | 特殊道具 |
| 📚 知识之书 | 永久 +5% 专注效率 | 200 | 永久升级 | 成长系统 |
| 🧠 冥想之石 | 永久 -10% 精力消耗 | 250 | 永久升级 | 成长系统 |

**刷新机制**（参考 Battlegrounds）：
- 每次商店刷新 4 件物品
- 可花 5 专注币刷新（限 2 次/轮）
- 刷新还是不刷新？这是个决策——和 The Bazaar 一样

#### 1c. 装备配置（30s）

```
当前装备栏（最多3个装备位）：
  ┌──────┐ ┌──────┐ ┌──────┐
  │  🛡️  │ │  ⏱   │ │  🔮  │
  │专注盾 │ │延长器 │ │预言晶│
  │(主动) │ │(主动) │ │(主动) │
  └──────┘ └──────┘ └──────┘

背包（未装备）：
  💊 精力药剂  ×2
  ⚡ 双倍卡    ×1
  📋 任务加速  ×3

[自动推荐装备]  [手动调整]
```

**策略点**：
- 装备位有限（3 格），每轮开始前重新分配
- 不同轮次需要不同策略：前期囤资源 vs 后期冲刺
- 是不是有点像卡牌游戏的"手牌管理"？

#### 1d. 任务设定（30s）

```
本轮学习目标：
  □ 高数第三章复习      ⭐ 常规任务   +10 专注币
  □ 英语单词 50 个      ⭐ 常规任务   +10 专注币
  □ 阅读论文 2 篇       ⭐⭐ 困难任务  +20 专注币
  ─────────────────────────────
  □ 额外挑战: 连续3天打卡  🔥 成就任务  +30 专注币
```

**策略点**：
- 任务难度自选——选简单但奖励少，还是选难但奖励多？
- 和 The Bazaar 的"选怪"一样：低风险低回报 vs 高风险高回报

#### 1e. 下注（10s，仅第一轮设置）

```
本局赌注：
  ○ 娱乐（10 币）  ○ 挑战（50 币）  ● 生死（200 币）
```

- ✅ 最底部一行"专注币累计收入"（极小字，淡灰）

### 阶段 3：复盘（1-2 分钟）

每轮专注结束后，展示本轮结果：

```
┌──────────────────────────────┐
│  Round 2 完成！              │
│                              │
│  本回合数据:                  │
│  专注时长: 25min ✅           │
│  中断次数: 0                  │
│  任务完成: 2/3                │
│                              │
│  专注币收入:                  │
│    基础:     +10             │
│    任务奖励:  +20            │
│    装备加成:  +15 (护盾奖励)  │
│    本回合:    +45 币          │
│                              │
│  对手动态:                    │
│  👤 影子 · 白银II             │
│  Round 1 完成 ▓▓▓▓▓░░░      │
│  总专注: 24min | 中断 1次     │
│                              │
│  目前总比分:                  │
│  你: ████████████  49:42     │
│  对手: ██████████░░           │
│                              │
│  [备战 Round 3 →]             │
└──────────────────────────────┘
```

**信息控制**：
- 能看到对手的进度（第几轮了）
- 能看到对手上一轮的数据（专注时长、中断次数）
- 但看不到对手的装备、策略、事件
- **适度信息**——知道自己在领先还是落后，但不清楚全部

这个"有限信息"制造了 **The Bazaar 式的悬念**——你知道个大概，但不知道全貌，最终结算才揭晓一切。

## 六、和经济系统的联动

### 专注币在远征中的流动

```
远征开始：余额 100 币

Round 1:  -25(买护盾)   +35(专注收入)  = 110
Round 2:  -55(买装备+刷新) +55        = 110
Round 3:  -15(买药水)   +25(扣中断)   = 120
Round 4:  -75(买水晶+锁定+延长) +80  = 125
远征结算: +100(赌注)                   = 225

→ 远征前后：100 → 225（净赚 125）
→ 但如果输了赌注，就只剩 125（净赚 25）
→ 有风险有回报
```

### 赌注的权重

```
娱乐局(10币)：赢了 +10，输了 -10 → 心理负担低
挑战局(50币)：赢了 +100，输了 -50 → 有感觉了
生死局(200币)：赢了 +400，输了 -200 → 心脏受不了

专注币的主要获取方式还是专注本身
赌注只是"让胜负有点分量"
```

## 七、策略维度总览

```
每轮备战的决策点：

1. 事件应对
   - 正面事件：如何最大化收益
   - 负面事件：如何降低损失
   - 抉择事件：选哪条路

2. 商店决策（经济博弈）
   - 买什么装备？→ 本回合策略取向
   - 刷新不刷新？→ 花 5 币赌一个更好的选择
   - 存钱还是花钱？→ The Bazaar 式取舍

3. 装备配置（3格限制）
   - 攻击型（延长器+目标锁定） vs 防御型（护盾+药剂）
   - 侦察型（预言水晶） vs 经济型（双倍卡）

4. 任务选择（难度博弈）
   - 简单任务多 → 稳但收益低
   - 困难任务 → 高风险高回报
   - 平衡组合 → 中庸之道

5. 对手信息解读（信息不对称）
   - 对手数据透露了什么？
   - 对手买了什么装备？能推断他的策略吗？
   - 最后一轮怎么根据对手布局调整？
```

**一场远征 = 至少 20 个策略决策**（4 轮 × 5 个决策点）。这还叫"太简单"吗？😏

## 八、和学习工具核心价值的平衡

### 游戏化 vs 专注的时间配比

```
一场 4 轮远征的时间预算：

  备战总计:  4轮 × 2min   = 8min        ← 游戏化时间
  专注总计:  4轮 × 25min  = 100min      ← 学习时间
  复盘总计:  4轮 × 1min   = 4min        ← 轻度游戏化
  终局结算:  1次 × 2min   = 2min        ← 游戏化高潮
  ─────────────────────────────────
  总时间:                  ~114min
  游戏化占比:               ~12%
  纯学习占比:                ~88%
```

88% 的时间是纯粹的学习。12% 的时间用游戏化驱动这 88% 的效率。

### 什么时候做这些"游戏操作"？

```
备战操作不是独立打开的 APP 时间
而是：
  • 番茄钟之间的休息时间（5min 休息）
  • 原本你会刷手机 → 改为做策略决策
  • 这个"置换"本身就是产品价值

每轮 25min 专注结束后，5min 休息
其中 2min 做备战决策，3min 喝水/活动
  
→ 游戏化不是占用学习时间
→ 游戏化是占用了休息时间的刷手机时间
```
