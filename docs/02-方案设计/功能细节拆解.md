# 功能细节拆解：人文松组队规则与角色分工

## 一、功能基本信息

- **功能模块**: 组队与分工
- **优先级**: P0（活动核心流程）
- **用户价值**: 让参与者在有限时间内高效组队并明确分工

---

## 二、组队规则

### 2.1 组队方式

| 规则项 | 内容 |
|--------|------|
| 方式 | **自由组队** |
| 流程 | 开幕公布主题 → 个人独立思考 → 每人分享选题切口 → 兴趣聚合自然组队 |
| 跨学科要求 | **建议跨学科**（软性引导），不强制 |

### 2.2 人数与限制

| 规则项 | 内容 |
|--------|------|
| 每队人数 | **3-4人** |
| 最少/最多 | 允许**单人成队**，上不设硬上限（但建议不超过4人） |

### 2.3 异常处理

| 场景 | 处理方式 |
|------|---------|
| 有人落单 | 允许单人成队，也可主办方协助协调 |
| 队伍人数过多（>4人） | 建议拆分成两队，或保留但提示效率风险 |
| 队内学科过于单一 | 软性建议对比其他队伍结构，不强制调整 |

### 2.4 组队环节时间线

| 时间 | 环节 |
|------|------|
| 14:00-15:00 | 个人独立思考——写"我想探索的切口"（200字以内） |
| 15:00-16:00 | 每人1分钟分享切口，兴趣聚合组队 |
| 16:00-17:00 | 队内深入讨论：定选题、定角度、定分工 |

## 三、团队角色分工

### 3.1 角色总览

| 角色 | 职责 | 是否需要特定技能 | 是否必须 |
|------|------|----------------|---------|
| 方向/点子 | 提出选题方向、概念定位、确定叙事角度 | 不需要 | ✅ |
| 素材采集 | 采访/拍照/录音/查资料/整理素材 | 不需要 | ✅ |
| 内容产出 | 写作/叙事/内容落地/文本组织 | 不需要 | ✅ |
| 视觉/排版 | 排版/配图/展示设计 | 不需要 | ❌ 可无 |

### 3.2 角色详解

#### 角色A：方向/点子

**核心任务**：
- 在开放大主题下找到具体的切入角度
- 确定"我们要做什么、为什么要做这个"
- 把控产出的核心观点和叙事方向

**哪里需要"技能"**：不需要。每个人都在自己学科里受过"找问题"的训练。

**关键时刻**：组队讨论阶段（Day1 16:00-17:00）、创作中途选题走偏时需要拉回。

**常见问题**：想法太多收不住 → 需要配合"牧羊犬"类型的人来收敛。

---

#### 角色B：素材采集

**核心任务**：
- 采访相关人、收集口述/故事
- 拍摄照片/视频素材
- 查阅文献/档案/数据
- 整理素材供内容产出使用

**工作方式**：视选题而定——可能是外出做田野，也可能是案头研究。灵活调整。

**产出**：原始素材包（音频/照片/笔记/数据表）。

**关键时刻**：创作前期（Day1 晚上 → Day2 上午）。素材够了才能写。

---

#### 角色C：内容产出

**核心任务**：
- 将素材组织成叙事
- 负责产出的主要文本/内容
- 把控作品的结构和语言

**注意**：产出形式不一定是"写文章"。如果是做展览，这个角色就是策展叙事；如果是做田野档案，就是整理成文。

**关键时刻**：Day2 上午集中创作期。

---

#### 角色D：视觉/排版（可选）

**核心任务**：
- 为产出做视觉呈现（排版/配图/展示设计）
- 路演展示材料的视觉辅助

**门槛**：不要求设计能力。Canva/PPT/Word排版即可。甚至可以在团队中不存在这个角色。

**关键时刻**：路演前冲刺阶段（Day2 下午）。

## 四、团队搭配建议

### 4.1 理想配置示例（4人队）

| 角色 | 背景举例 |
|------|---------|
| 点子王 | 哲学系/中文系——擅长提问题 |
| 素材采集 | 人类学/社会学——擅长找素材 |
| 笔头 | 中文系/新闻系——擅长写 |
| 视觉/排版 | 有需要可有人兼任，或从缺 |

### 4.2 最小配置（3人队）

角色A+B+C 即可运行，角色D可有可无。

### 4.3 理论 vs 实践的平衡

不设硬性要求。但建议团队在组队时自己讨论：
- "我们是想做一个偏概念/思辨的作品，还是偏记录/实证的作品？"
- "我们的素材从哪里来？需要采访吗？需要外出吗？"

这决定了素材采集角色的工作强度和时间安排。

## 五、参与者报名时的信息收集

为了辅助组队环节，报名表可收集以下信息：

1. **学校/专业/年级**
2. **一句话描述你感兴趣的人文议题方向**
3. **你希望在这个团队中扮演什么角色？**（可参考上面的角色列表，也可自定义）
4. **有没有特别想合作的专业方向？**（可选）
5. **你擅长的产出形式？**（文字/影像/声音/策展/游戏/其他）

这些信息可以帮助组队环节更快地让相似方向的人找到彼此。
