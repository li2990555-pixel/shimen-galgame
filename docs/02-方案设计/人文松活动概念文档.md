## 一、核心理念

### 一句话定义

> 人文松是一场以**人文学科各专业独有的表达方式**为媒介的限时组队创作活动。

### 核心三要素

| 要素 | 内容 |
|------|------|
| **限时** | 继承黑客松的限时挑战机制，在固定时间内完成创作 |
| **跨学科** | 不同人文社科背景的参与者组队协作 |
| **有效产出** | 产出是"可以被别人体验/展示的东西"，而非论文或漫谈 |

### 不是追求"创新"，而是追求"人文"

人文松继承了黑客松的形式，但不继承黑客松的价值取向。技术领域的评价标准是"新不新"——新的架构、新的功能、新的突破。但人文学科的价值不在这里。

> 人文松追求的是**人文深度、历史厚度、感受的准度**，而不是原创或创新。

一篇关于菜市场的口述史，方法并不新，但它忠实记录了一群人的声音，这就是人文的价值；一件关于《论语》当代解读的作品，没有"创新"的学术贡献，但它回应了千年传统中的某个回响，这就是人文的价值。

**"创新"不应该是人文松的评判标准。技术的逻辑是"向前"，人文的逻辑是"向下"和"向内"。** 这一定位区分了人文松与所有"人文主题的技术黑客松"的根本差异。

### 人文松要解决的核心问题

年轻人（尤其是人文社科背景）有表达欲和观察力，但缺少一个**低门槛、高密度、产出导向**的表达场合。现有的活动形式都有局限：

| 现有形式 | 局限 |
|---------|------|
| 学术会议/论文 | 门槛高、周期长、受众窄 |
| 写作工作坊 | 缺乏限时压力和完成感 |
| 沙龙/读书会 | 停留在讨论层面，无实质产出 |
| 艺术展/策展 | 需要专业美术/策展经验 |
| 技术黑客松 | 需要编程能力 |

人文松填补的是这个空白。

## 二、与落地松/技术黑客松的区别

这是人文松最核心的定位——它不是一个"人文主题的技术黑客松"。

| 维度 | 技术黑客松 | 落地松（2015） | **人文松** |
|------|-----------|---------------|-----------|
| **核心媒介** | 代码 | 代码+硬件+艺术 | **人文学科各专业的表达方式** |
| **产出形式** | 可运行的app/网站 | 多媒体装置/互动原型 | **展览/田野档案/影像/表演/游戏/叙事/行为设计等** |
| **所需技能** | 编程/设计/硬件 | 编程+设计+艺术 | **各人文学科自有方法论** |
| **人文的角色** | — | 作为活动的主题/题材 | **同时作为主题和方法** |
| **技术的位置** | 核心，不可替换 | 核心，不可替换 | **工具之一，可替换（允许但不必须）** |

**关键区分**：
- 落地松 = **用技术和艺术的手段** 去回应一个人文主题
- 人文松 = **用人类学做田野、用电影做纪录片、用哲学做概念设计、用中文系做叙事**——用各自的学科"母语"去表达

## 三、活动框架

### 基本参数

| 参数 | 首场方案 |
|------|---------|
| 时长 | 1.5天（周六下午 → 周日下午） |
| 规模 | 10-20人，每队3-4人 |
| 主题机制 | 半开放命题（给定大方向，各队自选切入角度） |
| 组队方式 | 兴趣驱动（先各自思考选题，再根据方向聚合成队） |
| 场地 | 优先学校免费空间 |
| 预算 | 0-1000元 |

### 产出形式：最小产出单位

**核心原则：人文松不无限开放。** 每队必须从以下四种"最小产出单位"中选择一种作为最终交付：

> 产出必须是**你所在学科的表达方式**，而不是通用写作或PPT。

| 最小单位 | 解释 | 适合的学科方向 |
|---------|------|--------------|
| **一个可体验的叙事** | 让人沉浸于一个故事——文本、影像、声音，或三者的混合 | 中文/文学、电影/传媒、新闻、播客 |
| **一个可浏览的档案** | 记录一群人/一个地点/一段时间的痕迹——田野笔记、口述史、物证收集 | 人类学、社会学、历史、策展 |
| **一个可参与的规则** | 进入一个由规则构成的世界——概念装置、辩论设计、纸面游戏、行为规范 | 哲学、游戏设计、戏剧/表演 |
| **一个可观看的表演** | 在特定时空发生的一次性呈现——行为、即兴、声音演出、戏剧片段 | 表演/戏剧、音乐/声音、舞蹈 |
**每一项产出必须能在最后的路演环节被展示或体验。** 叙事可以朗读或放映，档案可以翻阅，规则可以现场参与，表演直接呈现。

**这四种单位不限制学科表达**——同一个"可浏览的档案"可以由人类学团队做田野档案，也可以由策展团队做微型展览，表达方式完全不同，但出口形式一致。这使得路演环节有结构上的对等性，便于观众理解和比较。

### 常见误区与预防

人文社科学生最容易滑入的三种无效产出，足以让活动死亡：

| 误区 | 表现 | 为什么危险 |
|------|------|-----------|
| **做读书会** | 讨论了一本书/一个概念，最后产出是一组讨论心得 | 无实质产出，变成沙龙 |
| **做论文汇报** | 写了一段学术分析，路演时念论文 | 回到了学术惯性的安全区 |
| **做PPT** | 把想法做成了演示文稿，没有其他形式 | PPT是辅助工具，不是产出本身 |

**三层预防：**

**第一层：预期过滤（招募阶段）**
- 文案明确写："这不是读书会，不是论文汇报，不是PPT展示"
- 给出"错误示范 vs 正确示范"对比——例如"做一篇城中村口述史分析 → 不对；做一份城中村居民的社区档案 → 对"

**第二层：规则阻断（活动现场）**
- 开幕时主持人强调："如果你们队的最终产出是PPT，那这个活动对你们来说就失败了"
- 组队结束后各队提交"产出形式确认"，主办方审核拦回无效形式

**第三层：环境引导（物理空间）**
- 投影仪仅用于播影片，不投PPT。现场提供纸、卡纸、胶带、马克笔、录音设备——物理材料优先，数字工具退后
- 允许PPT作为辅助（如路演时展示素材、放一段视频），但**产出本身必须是叙事/档案/规则/表演中的一种**

### 跨队回应机制（Remix）

**规则**：每队在路演时额外展示一张"回应页"，回应另一队的素材。

**流程**：
- 路演前1小时，各队随机抽取另一队的原始素材包（照片/录音/笔记/访谈稿等）
- 从中选择1-2件素材，用自己学科的方式写/画/录一段回应（约20分钟）
- 路演时展示原作素材 + 自己的回应，一页投影片即可

**目的**：让作品之间产生对话关系，路演不再是5个孤立展品，而是可见的学科碰撞。同时保持极低成本——不设独立创作时段，仅占用路演前最后一小时和路演中的一页展示。

**可选升级**：路演现场设"素材交换桌"，各队打印/导出素材开放自由翻阅，有兴趣可贴纸条回应——纯彩蛋环节。

### 与技术（AI/代码）的关系

- 代码/AI不是活动的核心
- 允许作为工具使用（如用AI辅助做图、用简单代码做互动原型）
- 但不强制，也不以此评判产出质量

### 参与者立场声明：不割席，以产出形式过滤

人文松不与已有黑客松划清界限，也不拒绝有技术背景的参与者。原则只有一条：**最终产出必须是人文社科各学科的表达方式，而不是一个技术产品。**

这意味着：
- **技术人欢迎参加**，但需要用他学科背景之外的表达方式来创作（例如用社会学方法做田野，或用电影语言做纪录片）
- 如果他想做的是一个App或网站，那这个活动不适合他——这不是他的场子
- 但如果他用编程能力帮队友做了一个互动原型中的小工具，那完全ok

**过滤机制不是查身份，而是看最终上台展示的是什么。** 招募文案把产出形式说清楚，自然会让不匹配的人自己离开。留下来的技术背景参与者，反而可能成为跨学科协作中的独特贡献者。

### 团队角色分工

| 角色 | 职责 | 是否需要特定技能 | 是否必须 |
|------|------|----------------|---------|
| **方向/点子** | 提出选题方向、概念定位、确定叙事角度 | 不需要 | ✅ |
| **素材采集** | 采访/拍照/录音/查资料/整理素材 | 不需要 | ✅ |
| **内容产出** | 写作/叙事/内容落地/文本组织 | 不需要 | ✅ |
| **视觉/排版** | 排版/配图/展示设计（可选） | 不需要 | ❌ 可无 |

#### 角色A：方向/点子

**核心任务**：在开放大主题下找到具体的切入角度，确定"我们要做什么、为什么要做这个"，把控产出的核心观点和叙事方向。**不需要**特定技能——每个人都在自己学科里受过"找问题"的训练。

**关键时刻**：组队讨论阶段（Day1 16:00-17:00）、创作中途选题走偏时需要拉回。

**常见问题**：想法太多收不住 → 需要配合收敛型角色来平衡。

#### 角色B：素材采集

**核心任务**：采访相关人、收集口述/故事、拍摄照片/视频素材、查阅文献/档案/数据、整理素材供内容产出使用。工作方式视选题而定——可能是外出做田野，也可能是案头研究。

**产出**：原始素材包（音频/照片/笔记/数据表）。

**关键时刻**：创作前期（Day1 晚上 → Day2 上午）。素材够了才能写。

#### 角色C：内容产出

**核心任务**：将素材组织成叙事，负责产出的主要文本/内容，把控作品的结构和语言。注意：产出形式不一定是"写文章"——如果是做展览，这个角色就是策展叙事；如果是做田野档案，就是整理成文。

**关键时刻**：Day2 上午集中创作期。

#### 角色D：视觉/排版（可选）

**核心任务**：为产出做视觉呈现（排版/配图/展示设计），路演展示材料的视觉辅助。**不要求设计能力**，Canva/PPT/Word排版即可。甚至可以在团队中不存在这个角色。

**关键时刻**：路演前冲刺阶段（Day2 下午）。

#### 团队搭配建议

**理想4人队配置示例**：

| 角色 | 背景举例 |
|------|---------|
| 点子王 | 哲学系/中文系——擅长提问题 |

## 四、主题候选方向

首场活动需要一个半开放的大主题。以下候选方向来自人文社科的经典概念，兼具学术深度和日常共鸣：

| 主题 | 吸引力 | 特点 | 子方向举例 |
|------|--------|------|-----------|
| **附近** | 高 | 门槛低，每个人都有自己的"附近"，延展空间大 | 消失的附近、附近的陌生人、感官地图 |
| **边界** | 高 | 当代最具张力的概念之一，多种切入角度 | 迁徙与边界、算法围栏、越界者 |
| **记忆** | 高 | 经典母题，连接私密与宏大 | 遗忘的技术、废墟考古、数字亡灵 |
| **他者** | 高 | 触及差异、边缘、共情，与当下高度相关 | 凝视与反凝视、怪物与异类 |
| **主体性** | 中高 | 学术味重，需要"翻译"为亲和标题 | 数字自我、边缘之声、AI时代的主体性 |
| **日常** | 中 | 门槛低，素材无处不在 | 被忽视的劳动、重复的仪式、微抵抗 |
| **物** | 中 | 物质文化方向，适合策展/装置类产出 | 物的传记、垃圾本体论 |
| **游戏** | 中 | 与活动形式形成镜像 | 严肃游戏、规则之外 |
| **声音** | 中 | 冷门但有爆发力，产出独特 | 声景、沉默的政治 |

## 五、参考案例：落地松（2015）

- **主办**: 中国美术学院跨媒体艺术学院·网络社会研究所
- **主题**: "新作一个杭州人"
- **时长**: 2天
- **形式**: Day1上午公开演讲（6场）→ Day1下午快速提案+组队+通宵创作 → Day2创作+演示
- **核心理念**: 用技术和艺术手段"重新/创新"一个杭州人
- **与本项目的区别**: 落地松本质上是"技术黑客松+人文主题"，而人文松追求的是"人文学科自己的创作形式+人文主题"

落地松的价值在于它证明了"人文场景+黑客松形式"的可行性，同时它也成为一个清晰的参照系，让人文松的差异化定位更加明确。
