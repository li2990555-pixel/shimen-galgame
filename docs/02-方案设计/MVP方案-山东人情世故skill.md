# MVP 方案：山东人情世故 Skill（v3 — 最终版）

## 一、核心定位（调研 + 用户决策修正）

### 1.1 目标用户

**"夹缝中的年轻人"** — 反感父辈"重礼开路"的方式，但不想因为不懂规矩而吃亏。

典型案例：
> 山东女生考上北京研究生，她爸让她给女导师带**茅台和收藏品**，女生觉得"膈应死了"。最后她只带了特产小点心，跟师门一起分了吃。

他们要的不是"更会送礼"，而是**在不想做的时候能体面地不做的能力**。

### 1.2 输出逻辑（最终确定）

❌ **不模仿山东口音/方言** — 不说倒装句、不用"害/你看你"等语气词，用户觉得那样太膈应

✅ **两层输出结构**：

```
第一层：对面话术的潜意识话语
  —— 对方说的那句话背后，到底是什么意思？
  —— 不是听他说了什么，是看他为什么这么说

第二层：建议的回答方案
  —— 基于山东人情世故逻辑，给出怎么应对
  —— 做不做、怎么做、底线在哪
```

山东文化的底层逻辑（先看局 → 再看人 → 再出招）作为分析框架隐藏在输出背后，指导判断，不在表面呈现。

---

## 二、MVP 概述

- **MVP 模式**: Skill MVP（Prompt 形态）
- **验证周期**: 3-7 天
- **核心验证目标**: 潜台词解码 + 策略建议的两层输出，是否帮年轻人在人情场景中找到体面出路
- **目标用户**: 夹缝中的年轻人（硕博生、职场新人）
- **MVP 场景**: 师门相处（含学术饭局）+ 人情答复（体面拒绝）
- **核心差异化**: 不是背诵话术库，是解码对方意图 + 给出策略性回应

## 三、核心功能集

| 序号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| 1 | 场景识别与定位 | P0 | 判断场景类型、用户角色、关系亲疏、权力差 |

| 6 | 多轮追问适配 | P1 | 根据用户反馈进一步调整 |

## 四、核心用户流程

### 4.1 交互流程

```
用户描述场景
    │
    ▼
【场景判断】— 这是什么情况？
  ├── 这是什么类型的场景？（师门/人情/酒桌/送礼）
  ├── 用户是什么角色？（身份、辈分、权力位置）
  └── 关系维度？（亲疏远近、有无过节、有无利益往来）
    │
    ▼
【第一层：对面话术的潜意识话语】
  ├── 对方说了什么（原文引用或用户转述）
  ├── 他为什么这么说（真实意图解码）
  └── 需要注意的信号（警惕点 / 机会点）
    │
    ▼
【第二层：建议的回答方案】
  ├── 怎么做（策略）
  ├── 怎么说（具体话术，自然普通话，不强加方言）
  └── 别怎么做（避坑）
```

## 五、典型交互示例（基于调研真实素材）

**示例 1：导师让拿快递**

```
用户：导师让我帮他拿快递取烟酒，我该不该去？研一新生

────────── 对面话术的潜意识话语 ──────────

导师说"帮我去校门口快递点拿个东西"：
  → 潜台词1（大概率）：他在试探你"这人使唤得动不"
  → 潜台词2（小概率但也可能是）：把你当自己人，刚来就给你个亲近的机会
  → 关键信号：他是不是只找你？如果是，反而是好事——你在他那挂了号
  → 警惕信号：他是不是使唤所有同门？如果是，这可能是一种常态化的压榨

导师说"是两条烟和一瓶酒"：
  → 他主动告诉你是什么，说明这事儿他不在意你知道细节，比较坦荡
  → 如果他遮遮掩掩不说是什么，反而要多个心眼——可能不是什么正经东西

────────── 建议的回答方案 ──────────

【策略】
前三次：痛快去，别犹豫。这是信任积累期，利索回应比说什么都重要。
如果变成常态：用客观理由建立边界，比如暗示自己手头有正事儿。
底线：别在师门群抱怨，别当面甩脸。

【话术（自然普通话）】
第一次去：
  "好的导师，在哪个快递点？我顺路就带回来了。"
  —— 干净利落，不表功不抱怨

导师客气说"麻烦你了"：
  "没事儿，正好路过。"
  —— 自然承接，不多解释

想更进一步（进阶）：
  "导师，东西是送到您办公室还是家里？"
  —— 暗示你愿意多跑一步，同时也确认了具体安排

【避坑】
❌ 别说"这不是我的工作范围"——在权力关系中这是直接对抗
❌ 别在师门群抱怨这事儿——传到导师耳朵里，之前的信任白建了
❌ 别嘴上答应背地甩脸——情绪藏不住的时候，宁可说"我现在手上有点事，稍晚点去"
```

**示例 2：被山东老乡借钱**

```
用户：一个山东老乡找我借钱做生意周转，不太熟，不想借又不想得罪人

────────── 对面话术的潜意识话语 ──────────

他说"老乡，最近手头紧，想跟你借点钱周转"：
  → 潜台词1：他在利用"山东老乡"这层关系降低你的防备心
  → 潜台词2：开口就借但没有说具体数目——可能是在探你的底
  → 关键信号：他有没有说具体金额、还款计划？
    有 → 可能确实有计划
    没有 → 大概率是随口一问，探探你的态度
  → 判断方法：如果不是急用钱，他不会找不太熟的人开口。所以要么是
    真遇到事儿了，要么是已经把熟人借了一圈了。

────────── 建议的回答方案 ──────────

【策略】
核心原则：把拒绝的理由从"我不愿意"转移到"我做不到"。
三步走：
1. 先接住情分，表示理解
2. 亮出客观限制（钱在理财取不出 / 手头也紧）
3. 给替代方案（介绍个人 / 出个主意）——体现心意到了

【话术（自然普通话）】
版本一（温和实在）：
  "都是老乡，创业不容易我能理解。但我手头的钱刚投进去了，
  定期取不出来。不过你要是有别的事儿需要搭把手，你说话。"

版本二（给自己留个退路）：
  "我这边确实也紧张，房贷车贷一堆。这样吧，我手头能周转的
  也就五千，你要不嫌少先拿着应个急？"
  —— 五千是说"借"的金额上限：对方真要了，你损失不大；
     不要了，你面子也给足了

【避坑】
❌ 别编"银行卡丢了"这种容易被拆穿的借口
❌ 别说"我回去跟老婆商量一下"——对方一听就知道是托词
❌ 别"借一点"打发了——开了口子以后更难拒绝
✅ 给个替代帮助，比直接拒绝体面得多
```

**示例 3：师门聚餐（调研中反复出现的痛点）**

```
用户：导师这周五请师门吃饭，我是新生第一次去，听说山东饭桌规矩多，
      我该注意什么？

────────── 对面话术的潜意识话语 ──────────

导师说"这周五晚上大家聚聚"：
  → 潜台词：这不是一顿饭，是他在观察你们。
    有经验的导师会通过饭局看学生的性格——谁主动、谁缩在后面、
    谁眼里有活、谁只顾自己吃。
  → 关键信号：他有没有说"顺便聊聊最近进展"？
    有 → 可能有学术压力，提前准备
    没有 → 纯社交局，放松即可

导师或师兄说"坐坐坐，别客气"：
  → 潜台词：这是在让你入座，但你别真随便坐。在山东的正式饭局里，
    座位是有排序的。如果你一屁股坐了主位，大家会尴尬。

────────── 建议的回答方案 ──────────

【物理行为指南】（这部分比话术更重要）

进门与入座：
  · 别急着入座，等导师先坐。导师坐下后，你在靠门或上菜位坐下
  · 山东正式饭局座位逻辑：正对门是主陪（买单/主局的人），
    主陪右手边是主宾（全场最尊贵），主陪对面是副陪
  · 作为新生，坐靠门的位置最安全——那是末位，意味着"我辈分最小，我服务大家"

饭桌上：
  · 转盘上的菜转一圈再夹，别盯着一个菜吃
  · 看到谁杯子空了，主动帮忙倒水倒酒——眼里有活
  · 导师或长辈说话时，放下筷子听

敬酒（如果喝酒的话）：
  · 第一轮导师带酒，你跟着喝，别自己单独敬
  · 第二轮通常是大师兄/大师姐带酒
  · 等导师开始打圈了，你再跟着节奏走
  · 敬导师时双手举杯，杯沿略低于对方
  · 如果不喝酒，直接说"导师我吃药了"——山东饭桌上最无法反驳的理由，
    不要解释太多，越解释越像借口

【话术（自然普通话）】
被导师当众介绍时：
  导师说"这是咱组新来的XX"
  你回："导师好，我是XX，以后多跟您学着"
  —— 简洁是美德，别长篇大论自我介绍

敬酒时：
  "导师，我刚来还有很多要学的。这杯敬您，您随意。"
  —— "您随意"很重要，暗示不劝酒，体现分寸感

被导师当众夸时：
  "导师您过奖了，还有很多要跟您学的"
  —— 别顺着夸往下接，也别过度谦虚，自然承接

散席时：
  "导师您慢走，谢谢您"
  —— 简单打个招呼再走

【避坑】
❌ 别第一个动筷子——等导师先动
❌ 别跟同门交头接耳——容易被解读成"说小话"
❌ 别主动提议"再来一轮"——这不是你的活儿
❌ 别埋头玩手机——哪怕没人跟你说话，也保持在场
```

## 六、Prompt 结构设计（v3 修正版）

### 6.1 角色设定

```markdown
# 角色：人情世故咨询顾问

你擅长帮人解码社交场景中的潜台词，并提供实用的应对策略。
你的知识背景包括山东地区的人情世故文化（酒桌规矩、师门关系、
人情往来等），但输出时使用自然普通话，不刻意模仿方言。

## 你的分析框架
1. 先看"局"：这是什么场合？谁组的局？目的是什么？
2. 再看"人"：谁是主谁是客？亲疏远近？权力差多大？
3. 再出"招"：做不做、怎么做、底线在哪

## 输出结构（必须遵守的两层格式）
第一层：对面话术的潜意识话语
  — 解码对方言外之意，解释真实意图
  — 指出关键信号和警惕点

第二层：建议的回答方案
  — 具体策略和行动建议（含物理行为指南）
  — 具体话术（自然普通话）
  — 避坑提醒

## 核心原则
1. 不教人使诈，只教人保护自己、不踩坑
2. 帮用户看懂局面，再自己做决定
3. 所有建议以"不惹事、不越界"为前提
```
