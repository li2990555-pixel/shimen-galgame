# 从 The Bazaar 看卷王对决的竞技设计

> 参考游戏：The Bazaar（Reynad / Tempo Storm）
> 核心命题：异步自走棋的竞技感如何应用到学习对战？

---

## 一、The Bazaar 的核心哲学

The Bazaar 的突破性在于：**它是异步的，但竞技感不输同步游戏。**

```
传统认知：异步 = 缺少紧张感，像回合制
The Bazaar：异步 ≠ 不紧张，关键在于——

  1. 准备阶段的策略深度（Build 决策）
  2. 对手信息的不透明（信息差）
  3. 结果揭晓的仪式感（Reveal）
  4. 经济系统的博弈（取舍）
```

这个框架完美适用于学习对战——

**学习本身就是异步的**：你学你的，我学我的，但我们可以比谁学得更好。

## 二、映射：The Bazaar → 卷王对决

### 核心映射

```
The Bazaar                   卷王对决
─────────────────            ─────────────────
一天 (Day)                   一个番茄钟 (25min)
Build / 装备组合             学习计划（任务+时间安排）
PVE 打怪 → 获得装备          完成番茄钟 → 获得专注币
PVP 对决 → 和其他玩家比      对战 → 和好友比专注数据
血量 (HP)                    专注完成度 / 中断次数
经济系统 (金币管理)           专注币（赌注+奖励）
银币/铜币对手等级            段位匹配
天梯排名                     段位系统
面板 (Board)                 学习任务面板
```

### 关键机制的详细映射

**1. Day → 番茄钟**

The Bazaar 里一天一天推进，每一天你要决定买什么装备、怎么摆阵。对应到学习：

```
The Bazaar 的一天：
  选择 → 购买/出售装备 → 排列阵型 → PVE/PVP

卷王对决的一个番茄钟：
  设定 → 今天学什么 → 安排优先级 → 开始专注
```

**一天的循环：**

```
回合开始 → 决策 → 自动执行 → 结果 → 下一回合
   ↓        ↓        ↓        ↓       ↓
专注开始 → 设定任务 → 专注学习 → 结算 → 下一番茄钟
```

**2. Auto-Battle → 专注执行**

The Bazaar 最妙的设计是：**你摆好阵后，战斗是自动的**。同样，学习也应该如此——你定好计划后，执行就是专注本身，不需要再做任何"操作"。

```
The Bazaar 的战斗：
  Build 质量决定胜负 → 操作不介入 → 看结果

卷王对决的专注：
  学习计划质量决定效果 → 专注不被打断 → 看数据
```

**3. 对手 Ghost → 影子挑战**

The Bazaar 匹配的是其他玩家"某一时刻的阵容快照"。对应到学习：

```
你的番茄钟记录 → 生成一个"影子" 
                     ↓
其他用户可以和你的影子对战
                     ↓
赢了你的影子 = 比你当时更专注
                     ↓
这是异步的自我超越！
```

**4. 信息不对称 → 悬念**

The Bazaar 中你不知道对手在 build 什么，直到对决开始。对应到学习：

```
战前：
  • 你知道对手选的时长和赌注
  • 你不知道他今天状态如何、任务难度
  
战中：
  • 能看到对手专注/中断状态
  • 不知道他具体在学什么内容

战后（揭晓）：
  • 全面数据对比
  • "原来他今天学了5个小时？！"
```

这个信息差创造了 **悬念**——异步竞技的灵魂。

## 三、重新设计：以 Bazaar 为蓝本的异步竞技模式

### 模式名称：「Bazaar 竞技场」

```
异步竞技，不用同时在线，但竞技感拉满
```

### 完整流程

```
阶段一：备战（15-30s）
────────────────
选模式 → 定任务 → 下赌注 → 进入专注

阶段二：专注（25min / 45min / 90min）
────────────────
纯专注，无任何游戏元素
唯一看到的信息：对手的"专注状态"
  • 对手已专注：12min
  • 对手状态：专注中

阶段三：结算（10s）
────────────────
KO 动画 → 数据对比 → 段位变动 → 下注结算

阶段四：复盘 / 下注下一局
────────────────
看数据详情 → 调整策略 → 匹配下一个对手
```

### 和同步模式的关系

```
同步模式（好友对战）          Bazaar 竞技场（异步匹配）
─────────────────           ────────────────────────
和好友约好同时开始           和全服任意玩家比数据
实时看到对方状态             专注中只看自己的
社交驱动为主                 竞技驱动为主
适合：固定学伴               适合：想验证实力
```

**两者不冲突，互补覆盖不同场景：**

```
时间固定 + 有好友在线  →  同步好友对战 ← 主场景
时间灵活 + 好友不在    →  Bazaar 异步   ← 补充场景
想验证自己的进步       →  影子挑战       ← 单人场景
```

## 四、Bazaar 经济系统的借鉴

The Bazaar 的经济系统（金币、利息、刷新）是策略深度的核心来源。映射到学习：

### 专注币经济 2.0

```
主要收入：
  • 完成一个番茄钟  → +10 币
  • 赢得对战         → +下注金额×2
  • 连胜奖励         → +连胜场次×5
  • 每日签到         → +20 币
  • 赛季结算         → +段位×100 币

主要支出：
  • 对战下注         → 10/50/200 币
  • 购买皮肤/特效     → 500-2000 币
  • 复仇战加倍       → 赌注×2（输了的话）

利息机制（Bazaar 核心设计）：
  • 每保留 50 币 → 每天额外 +3 币利息
  • 鼓励积累，也鼓励消费——取舍

取舍决策：
  • 存币吃利息（稳妥） vs 下注翻倍（激进）
  • 买限定皮肤（收藏） vs 留着下注（竞技）
```

### 为什么需要经济系统

The Bazaar 证明：**没有经济博弈的异步竞技，只是一次性的比分板。**

经济系统给每个决策增加了"重量"：
- 下注 200 币带来的紧张感，远比"随便打打"强
- 看到对手下了 200 币，你就知道他认真了
- 赢了暴富，输了要重新积累——和 The Bazaar 的"赢了吃鸡、输了重开"一样的情绪曲线

## 五、策略深度的分层设计

The Bazaar 玩家能感受到自己的进步：从"随便买"到"知道怎么配合"。学习也应有同样的成长曲线。

```
初级策略（第1周）
  • 选个时长，开始学
  • 下注最小的
  • 赢不赢看运气

中级策略（第1个月）
  • 根据状态选时长（状态好选90min）
  • 理解"任务清单"的杠杆作用
  • 知道什么时候该冲、什么时候该休

高级策略（第3个月）
  • 精确管理专注节奏（25-5-25-5-90-15）
  • 根据对手历史数据调整策略
  • 经济管理：存币 vs 下注的动态平衡
  • 赛季策略：季初冲分 vs 季末收割
```

每个层次的玩家都能感受到"我在变强"——**不是运气变好，是决策质量变高**。

## 六、The Bazaar 给我们的 5 个核心启发

| 启发 | 应用到卷王对决 |
|------|--------------|
| **异步 ≠ 不紧张** | 通过信息不对称（对手数据不透明）创造悬念，结算时揭晓 |
| **准备比操作重要** | 战前定计划、选时长的决策质量决定胜负，而非学习时的"操作" |
| **经济系统是策略核心** | 专注币的收支决策（存 vs 花）提供持续的 meta 博弈 |
| **Ghost 是最佳异步机制** | 影子挑战——和过去的自己比，单人也能玩 |
| **Build 多样性 = 重玩性** | 不同专注模式（闪电/深潜/马拉松）+ 不同的任务策略= 排列组合多样性 |

## 八、MVP 的 Bazaar 模式规格

```
对战匹配：
  • 系统根据段位匹配实力相近的对手
  • 匹配到的是对手最近一次专注记录（类似 Ghost）
  • 匹配耗时 < 3s

对战展示：
  • 对手匿名（仅显示段位和胜率，不显示 ID）
  • 战中只显示百分比进度（不显示绝对时长，制造悬念）
  • 结算才揭晓所有数据

下注系统：
  • 自选赌注金额（10/50/200）
  • matches The Bazaar 的"风险-收益"取舍

反馈节奏：
  专注每 5min → 显示一次对手进度（百分比）
  完成番茄钟 → 一次"攻击"动画
  对手中断 → 静默记录，结算时展示
  结算     → 全面揭晓 + 段位变动
```
