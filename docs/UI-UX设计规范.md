# 卷王对决 — UI/UX 设计规范

> 版本：v1.0 | 日期：2026-06-18

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **能量感** | 用动态元素（进度条、粒子、动画）传递"战斗"氛围 |
| **专注优先** | 计时状态下 UI 极简，不干扰学习 |
| **游戏化节奏** | 每个操作都有即时反馈（声音+动效） |
| **社交张力** | 时刻让你看到对手的状态，制造压迫感 |

## 2. 品牌设计

### 2.1 Logo 与吉祥物

- **品牌名**：卷王对决
- **英文**：Study Clash
- **吉祥物**：一只戴着拳击手套的熊猫（"拳王"谐音"卷王"）
- **配色联想**：竞技红 × 专注蓝 × 晋级金

│                                    │

**设计要点**：
- **计时状态下是极简模式**——深色背景，只有计时器和对手状态
- 顶部进度条显示对手的完成度
- 番茄钟完成会触发全屏"攻击"动画（拳击熊猫出拳）
- 任务清单自动打勾（完成一个番茄钟勾一个任务）
- 暂停/结束按钮需要长按确认，防止误触

## 4. 关键交互细节

### 4.1 动效规范

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 页面切换 | 滑动进入 | 300ms | ease-out |
| 按钮点击 | 缩放 0.95→1.0 | 150ms | spring |
| 对战匹配 | 粒子聚集动画 | 800ms | ease-in-out |
| 专注完成 | 全屏冲击波 | 600ms | ease-out |
| 段位升级 | 彩带 + 闪光 | 1200ms | ease-out |
| 结算入场 | 格斗 KO 特效 | 1500ms | ease-out |
| 对手中断 | 脉冲闪烁 | 持续 | |
| 连胜提示 | 火焰燃烧 | 持续 | |

### 4.2 声音设计

| 事件 | 音效风格 |
|------|---------|
| 发起对战 | 战鼓声 |
| 专注开始 | 清脆铃声 |
| 番茄钟完成 | 升级音效 |
| 对手中断 | 警报声 |
| 结算胜利 | 胜利号角 |
| 结算失败 | 低沉呜声 |
| 段位升级 | 史诗管弦 |
| 按钮点击 | 轻脆 click |

### 4.3 触感设计（移动端）

| 操作 | 触感反馈 |
|------|---------|
| 开始专注 | 短震 |
| 番茄钟完成 | 长震 + 成功感 |
| 对手中断 | 短促脉冲 |
| 结算失败 | 低落触感 |
