# 卷王对决 — 竞技模式与游戏化参考系

> 关键问题：我们的对战到底是同步还是异步？参考的具体游戏是什么？

---

## 一、三幕式 ≠ 肉鸽，是"格斗游戏三段式"

### 和肉鸽（Roguelike）的区别

| | 肉鸽 | 卷王对决 |
|--|------|---------|
| 战中体验 | 核心操作+决策，**全程游戏** | **≈0 游戏化**，纯专注 |
| 战中随机性 | 随机地图/装备/BOSS | 无随机，对手专注度是唯一变量 |
| 战中反馈 | 实时数值+特效+音效 | 一条对手状态文字（被动查看） |
| 结算 | 死亡/通关，收获资源 | KO 动画+数据分析 |

**结论**：结构上确实有三段式，但"魂"完全不同。

### 更准确的参考系：格斗游戏的选人→对战→KO

```
格斗游戏:   选角色→放狠话 → 对战中(全神贯注) → KO! → 结算
                ↓               ↓                  ↓     ↓
卷王对决:   选模式→下赌注 → 专注中(全神贯注) → KO! → 数据复盘
```

我们抄的是格斗游戏的 **仪式感结构**——战前制造张力、战中排除干扰、战后释放爽感。这和格斗游戏"选人→对打→KO"的节奏完全一致。

### "肉鸽感"从哪来？

如果你觉得有肉鸽味，应该是来自 **"下注"的决策感** 和 **"专注倍率"的风险收益**——这两个设计借鉴了肉鸽的"build 决策"元素，但被压到了战前阶段，不影响专注。

## 二、核心问题：同步还是异步？

这是最关键的设计决策。两种方案对比如下：

### 同步模式（Real-time 1v1）

```
你 ───────────── 实时同步 ────────────→ 小明
  同时开始专注                        同时开始专注
  看到你状态                          看到你状态
  同时结束                            同时结束
  ↓                                    ↓
  └──────── 同时结算 ──────────────────┘
```

| 维度 | 评价 |
|------|------|
| 紧张感 | ⭐⭐⭐⭐⭐ 同时开战，压迫感最强 |
| 仪式感 | ⭐⭐⭐⭐⭐ "3-2-1 开始！"像格斗游戏 |
| 社交临场感 | ⭐⭐⭐⭐⭐ 知道对方也在学，心理驱动极强 |
| 时间协调 | ⭐⭐ 需要两人同时在线，门槛高 |
| 匹配难度 | ⭐⭐ 要双方都有空 |
| 覆盖面 | ⭐⭐ 对时间要求高，拒人门外 |

**参考游戏**：拳皇/街霸（1v1 格斗）、Clash Royale（3分钟同步对战）、王者荣耀（实时对抗）

**适合场景**：
- 和好友约定时间一起学（"8点来一把？"）
- 专注时间段固定的人（如每晚 8-10 点）
- 需要强外部压力才能专注的用户

### 异步模式（Async / Turn-based）

```
你 ─── 专注25min → 提交成绩 ──┐
                               ↓
                        系统缓存结果
                               ↓
小明 ─── 专注25min → 提交成绩 ──┘
              （72h 内完成）
```

| 维度 | 评价 |
|------|------|
| 紧张感 | ⭐⭐ 缺少实时对抗感 |
| 仪式感 | ⭐⭐⭐ 有胜负但没"同时开战"的张力 |
| 社交临场感 | ⭐ 不知道对方正在进行 |
| 时间协调 | ⭐⭐⭐⭐⭐ 随时可打，门槛极低 |
| 匹配难度 | ⭐⭐⭐⭐⭐ 无需协调，随时匹配 |
| 覆盖面 | ⭐⭐⭐⭐⭐ 覆盖所有用户 |

**参考游戏**：LearnClash（72h 异步 Quiz 对战）、Clash Royale 的异步匹配、自走棋类（摆阵后自动对战）

**适合场景**：
- 时间不固定的人
- 不想社交、只想比个成绩
- 全球匹配、扩大用户池

### 推荐方案：同步为主 + 异步补充

```
┌─────────────────────────────────┐
│        卷王对决 竞技模式          │
│                                 │
│  🥇 同步对战（主模式 80%精力）   │
│     ├── 好友对战                 │
│     ├── 排位匹配                 │
│     └── BOSS 战（合作）          │
│                                 │
│  🥈 异步模式（补充 20%精力）     │
│     ├── 影子挑战（vs 过去的自己） │
│     ├── 战书模式（下战书任一时间）│
│     └── 赛季榜（全异步排行）      │
└─────────────────────────────────┘
```

**为什么同步是主模式**：
1. 我们的核心价值是"**同时**"——Focus Town 证明了"看到别人在学习"是强大的驱动力
2. 双人好友场景本身就意味着"约定时间"，同步不是障碍而是仪式
3. 如果纯异步，和刷排行榜有什么区别？那就失去了"对战"的本质
4. 格斗游戏的灵魂就是**同步**——"我要当面击败你"

**异步做补充**：
- 好友不在线时，影子挑战让你有事做
- 下战书：给好友留一个挑战，他上线后接受并完成（类似 LearnClash）
- 赛季排行榜：自然的异步竞争

## 三、具体参考游戏拆解

### 核心参考系 1：格斗游戏（同步对战·拳皇/街霸）

```
我们抄什么：
✅ 选人界面 → 选模式/场馆（5-10s）
✅ 放狠话   → 战前互飙（3轮）
✅ Ready? GO! → 3-2-1 倒计时开战
✅ 对战中   → 专注计时（无干扰）
✅ K.O.!    → 全屏 KO 特效
✅ 结算     → 段位变动+数据复盘
✅ 复仇战   → 输了可立即复仇
✅ 连胜宣言 → 连胜有特殊标识
✅ 晋级赛   → 升段时需要打晋级赛

我们不抄：
❌ 连招系统 → 专注不需要操作技巧
❌ 血量博弈 → 专注度不依赖反应速度
❌ 帧数优势 → 和设备性能无关
```

### 核心参考系 2：Clash Royale（同步竞技场结构）

```
我们抄什么：
✅ 3分钟一场 → 25min 一个番茄钟
✅ 奖杯系统 → 段位分（赢了+xx，输了-xx）
✅ 皇冠宝箱 → 专注币奖励
✅ 赛季重置 → 每月赛季，段位重置
✅ 部落     → 学习小队（V3）
✅ 观战     → 好友对战时围观（V3）

我们不抄：
❌ 卡牌等级 → 不卖数值
❌ 氪金加速 → 专注币不能充值
❌ 开箱随机 → 奖励确定性
```

### 核心参考系 3：Focus Town（自习场景 + 实时在线感）

```
我们抄什么：
✅ 实时看到他人状态 → 对手专注状态显示
✅ 自习场景主题   → 对战场馆
✅ MVP 结算画面   → KO 结算

我们不抄：
❌ 陌生人自习室   → 我们专注双人好友
❌ 虚拟世界装修   → 我们专注竞技成长
❌ 3D 人物自定义  → V3 再做
```

### 核心参考系 4：Duolingo（连胜 + 段位驱动）

```
我们抄什么：
✅ 连胜机制    → 连胜加分加成
✅ 段位系统    → 同款 8 段位设计
✅ 每日签到    → 每日对战奖励
✅ 社交排行榜  → 好友榜

我们不抄：
❌ 红心（生命值）→ 限制学习次数不合理
❌ 宝石经济     → 专注币更简洁
❌ 课程闯关     → 专注不限内容
```

### 核心参考系 5：LearnClash（异步竞技)

```
我们抄什么：
✅ ELO 匹配 → 公平匹配
✅ 异步挑战 → 战书模式
✅ 段位体系 → 参考其段位命名

我们不抄：
❌ 纯异步 → 我们同步为主
❌ Quiz 答题 → 我们专注时长
```

### 核心参考系 6：魔兽世界竞技场（2v2 组队）

```
我们抄什么：
✅ 2v2 组队 → BOSS 战合作模式
✅ 竞技场赛季 → 赛季奖励/坐骑（皮肤）
✅ 决斗标志 → 好友头像上显示"可对战"
✅ 战歌 → 赛季主题音乐

我们不抄：
❌ 装备等级 → 不卖数值
❌ DKP → 过于复杂
```

## 四、竞品模式地图

```
                     同步
                      │
                      │
       Focus Town     │  格斗游戏
       (自习在线)      │  (拳皇/街霸)
                      │
        异步对战 ←────┼────→ 同步对战
                      │
                      │
       LearnClash     │  Clash Royale
       (异步Quiz)      │  (3min同步)
                      │
                      │
                    异步

我们的位置：右上象限 → 同步对战
补充位置：影子挑战（单机异步）→ 左上象限
```

**为什么选同步**：
- 右上象限（同步对战）是蓝海——Focus Town 在同步自习，但不对战；LearnClash 在对战，但是异步
- 同步的"同时感"是最大的心理驱动力——知道对方也在学，比你设 100 个番茄钟都有用
- 异步对战最终会变成"比谁的记录高"，社交临场感弱，容易弃用

## 五、MVP 竞技模式规格

```
同步对战流程（完整版）：

1. 发起方操作（15s）
   - 选择好友 → 选择模式（快速/深潜） → 设定赌注
   
2. 对方确认（10s）
   - 收到推送 → 看到详情 → 接受/拒绝
   - 拒绝后可选择"下战书"（异步）
   
3. 战前准备（15s）
   - 放狠话（可选） → 选择场馆 → 双方确认"准备就绪"
   
4. 倒计时（3s）
   - 3-2-1 GO! → 双方进入专注模式
   
5. 战中（25min+）
   - 大字计时器 + 对手状态（一行）
   - 休息间隔完成番茄钟 → 显示"攻击"反馈
   
6. 结算（10s）
   - KO 动画 → 数据对比 → 段位变动
   - 发战报 → 复仇战（输了可点）
```

### 时间预算

| 阶段 | 时长 | 游戏化强度 |
|------|------|-----------|
| 发起→确认 | ~30s | 🔥🔥🔥🔥 |
| 战前准备 | ~15s | 🔥🔥🔥 |
| 倒计时 | 3s | 🔥🔥🔥🔥🔥 |
| 专注中 | 25min | ❄️（≈0） |
| 结算 | 10s | 🔥🔥🔥🔥🔥 |
| **游戏化占比** | **< 4% 的时间** | **对专注影响≈0** |
