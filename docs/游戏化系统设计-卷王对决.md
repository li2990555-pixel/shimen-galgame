# 卷王对决 — 游戏化系统深度设计

> 关键命题：让对战足够刺激，又不干扰学习专注

---

## 核心设计原则

### 「三幕式」体验分割

```
战前 (30s)         战中 (25min+)             战后 (15s)
────────            ──────────                ──────────
🔥 全力游戏化        🧘 极致专注               🎉 全力游戏化
• 选对手·选模式      • 只有计时器              • KO 结算动画
• 下赌注·放狠话      • 对手状态(一行)          • 数据对比
• 选角色·配技能      • 番茄完成=攻击(动画)     • 升级·掉段
                    • 摸鱼检测(静默)          • 战利品
                    • ─── 任何游戏元素        • 连胜记录
                    • ─── 都会破坏专注        • 发战报炫耀
```

**关键洞察**：战中阶段要"反游戏化"——越少干扰越好。精彩全在战前和战后。

## 一、战前：制造紧张感

### 1.1 「下注系统」—— 把胜负变得有分量

双方在对战开始前各押注一定数量的 **专注币（Focus Coin）**。赢了全拿，输了全扣。

| 赌注档位 | 金额 | 视觉效果 |
|---------|------|---------|
| 🥉 娱乐局 | 10 币 | 普通边框 |
| 🥈 挑战局 | 50 币 | 银色边框 + 火花特效 |
| 🥇 生死局 | 200 币 | 金色边框 + 烈焰特效 |

**设计要点**：
- 专注币只能通过专注学习获得（完成番茄钟 = 赚币）
- 不能充值购买 → 保证公平性
- 输光后自动降回娱乐局，不可负债
- **"下注确认"需要双方点击**，制造仪式感

### 1.2 「放狠话」—— 赛前垃圾话

```
┌──────────────────────────────────┐
│  战前互飙                        │
│                                  │
│  小明: "今天你输定了 📚"         │
│  你:   "我昨天学了4小时 😎"      │
│                                  │
│  [发送]              [准备好了]   │
└──────────────────────────────────┘
```

- 限定 15 字以内，快速输入
- 预置狠话模板（"别学到一半睡着了"、"今天状态拉满"）
- 战前对话仅有 3 轮，不拖节奏
- 狠话会显示在结算页面的"战前语录"中，供赛后回味

### 1.3 「主题场馆」—— 不同的对战氛围

| 场馆 | 视觉风格 | 解锁条件 |
|------|---------|---------|
| 🏟️ 标准竞技场 | 默认 | 初始解锁 |
| 📚 深夜图书馆 | 暗色调 + 烛光 | 连胜 3 场 |
| 🌊 海滩自习室 | 蓝色系 + 波浪 | 累计专注 10h |
| 🏔️ 雪山之巅 | 冷色调 + 雪花 | 段位黄金以上 |
| 🚀 太空站 | 科幻风 | 赛季限定 |

**注意**：场馆只影响战前和战后的视觉，**专注计时界面始终是极简深色模式**。

## 二、战中：最小化游戏元素

### 2.1 专注计时 UI 原则

```
✅ 允许出现在专注界面的：
   • 计时器（大字）
   • 对手专注状态（一行文字 + 小绿点/灰点）
   • 本局任务清单（折叠状态）
   • 暂停/结束按钮（灰显，不抢眼）

❌ 禁止出现在专注界面的：
   • 段位分变动提示
   • 任何弹窗/通知
   • 动画/特效
   • 排行榜/成就
   • 商城/社交动态
```

### 2.2 「番茄冲击」—— 唯一的战中反馈

当用户完成一个番茄钟时（25min 到）：

```
专注完成瞬间的反馈：
1. 计时器显示 ✅ 完成（绿色）
2. 短暂的 0.5s 震动触感（手机）
3. 自动进入 5min 休息倒计时
4. 休息界面下方一行小字：
   "⚔️ 你对小明发动了一次专注攻击！"
   （仅文字，无动画）
```

**为什么这是必要的**：
- 让"完成番茄钟 = 攻击"这个映射刻入用户的认知
- 文字反馈 0.3s 即可读完，不延长休息时间
- 相比于纯番茄钟（没有外部反馈），增加了"每一分钟都在积累战斗力"的感觉

### 2.3 「摸鱼侦察」—— 静默的社会压力

当系统检测到对手可能中断时，专注界面底部会有一行微妙的变化：

```
对手状态变化:
  ● 专注中  →  正常绿色，安心
  ○ 休息中  →  灰色，正常休息
  ⚠️ 已中断 →  淡黄色（5min 无操作）
  🔴 已退出 →  红色，对局提前结束
```

**设计要点**：
- 不弹窗、不震动、不发出声音
- 仅在专注界面底部用一行小字显示
- 用户可以选择不看——因为专注时注意力在任务上
- 但如果瞟了一眼看到对手中断了，会产生微妙的"优势感"

### 2.4 「专注倍率」—— 最后的刺激

```
持续时间    倍率
────────────────
0-10min     ×1.0
10-20min    ×1.2
20-30min    ×1.5
30-45min    ×2.0
45-60min    ×3.0
```

- 专注越久，每分钟得分倍率越高
- 鼓励深度长专注，抵制碎片化
- 这个倍率**战中不展示**，只在结算时作为"惊喜"显示
- 让中途放弃的代价变得更大

### 3.2 「专注力数据分析」

| 指标 | 说明 | 游戏化包装 |
|------|------|-----------|
| 专注时长 | 总学习时间 | "攻击力" |
| 番茄钟数 | 完成完整周期数 | "连击数" |
| 中断次数 | 提前退出/暂停次数 | "受伤次数" |
| 全程度 | 是否完成所有番茄钟 | "满血通关" |
| 最大倍率 | 最长连续专注的倍率 | "暴击倍率" |
| 任务完成率 | 预设任务的完成比例 | "支线任务" |

```
            3 次                    ← 败者指标灰色

最高专注倍率  ×1.5 ⚡               ← 金色高亮
             ×1.0
```

## 四、元游戏系统（Meta Progression）

### 4.1 段位系统（竞技核心）

> 已在前序文档中定义，此处补充游戏化细节

**晋级赛机制**：
- 从白银升黄金需要打"晋级赛"（3 局 2 胜）
- 晋级赛期间界面有特殊边框和提示
- 晋级成功触发全屏晋级动画
- 晋级失败显示"差一点"鼓励文案

**保级机制**：
- 每个段位有"保级分"，长期不活跃会缓慢掉分
- 但专注学习可以积累"保级护盾"
- 1 小时专注 = 1 天保级护盾

### 4.2 专注币经济

```
赚取方式：
  完成番茄钟      +10 币/个
  对战胜利        +50 币
  连胜奖励        +连胜场次×10 币
  每日签到        +20 币
  赛季奖励        +100~500 币

消费方式：
  对战下注        10/50/200 币
  限定皮肤        500~2000 币
  特效            300~800 币
  表情包          100~300 币
```

**关键规则**：
- 专注币只能通过学习获得，不可充值
- 专注币余额 = "我有多努力"的证明
- 限制每日获取上限（防刷）：200 币/天
- 败方下注的币 80% 归胜方，20% 系统回收（抑制通胀）

### 4.3 成就系统

| 🏅 成就 | 条件 | 奖励 |
|---------|------|------|
| **初战告捷** | 完成第一场对战 | 头像框"初学者" |
| **三连胜** | 连续赢 3 场 | 50 币 |
| **十连胜** | 连续赢 10 场 | 限定特效"烈焰之路" |
| **满血通关** | 全程无中断获胜 | 100 币 |
| **马拉松选手** | 单次专注超 4 小时 | 徽章 + 200 币 |
| **早起卷王** | 早 6-8 点完成 10 场对战 | 称号"晨型卷王" |
| **深夜战神** | 晚 10-12 点完成 10 场对战 | 称号"夜行卷王" |
| **全能战士** | 3 种模式各赢 10 场 | 限定皮肤"全能斗士" |
| **最佳拍档** | 和同一好友对战 50 场 | 羁绊专属称号 |
| **百战卷王** | 累计完成 100 场对战 | 限定立绘"百战英雄" |
| **赛季王者** | 赛季结算段位王者 | 赛季限定皮肤 |
| **永动机** | 连续签到 30 天 | 特效"永恒之火" |

### 4.4 「英雄之路」—— 角色成长

每个用户有一个 **学习英雄** 角色，随着专注积累而成长：

```
等级  所需总专注    解锁内容
──────────────────────────
Lv.1     0h        初始形象
Lv.5     10h       新发型
Lv.10    50h       新服装 + 对战入场特效
Lv.20    200h      专属称号 + 结算动画
Lv.30    500h      传说皮肤
Lv.50    1000h     定制 KO 特效
Lv.99    5000h     终极形态
```

**英雄风格选择**（10h 后解锁）：

| 风格 | 描述 | 视觉 |
|------|------|------|
| ⚡ 闪电型 | 擅长短时爆发（25min 模式加成） | 蓝色闪电特效 |
| 🛡️ 壁垒型 | 擅长长时专注（马拉松模式加成） | 金色盾牌特效 |
| 🔥 火焰型 | 擅长连胜（连胜攻击加成） | 红色火焰特效 |

**风格不产生数值优势**，仅有视觉差异和称号差异——保证公平性。

### 4.5 赛季系统

```
周期：每月 1 日 → 月末

赛季主题示例：
  🥋 武道会赛季（格斗主题）
  🚀 星辰赛季（太空主题）
  🏯 忍者赛季（忍者主题）
  🧙 魔法赛季（魔法主题）

赛季内容：
  • 赛季限定任务（"本月累计专注20h"）
  • 赛季排行榜（好友榜 + 全区榜）
  • 赛季限定皮肤（过期绝版）
  • 赛季结算奖励（基于最终段位）

段位重置：
  赛季结束后，段位重置规则：
  王者   → 铂金I
  大师   → 黄金I
  钻石   → 白银I
  铂金以下 → 不变
```

### 4.6 羁绊系统

和同一好友多次对战后，羁绊等级提升：

```
Lv.1  一起打了 3 场   → 双倍周末活动解锁
Lv.2  一起打了 15 场  → 专属双人头像框
Lv.3  一起打了 30 场  → 组队入场特效
Lv.4  一起打了 60 场  → 羁绊专属称号
Lv.5  一起打了 100 场 → 双人限定皮肤
```

**特殊机制**：
- 羁绊等级高的两人对战，双方都有额外经验加成
- 高羁绊好友长时间不对战 → 羁绊会缓慢衰减
- 衰减可通过"双人组队"模式恢复

## 五、特殊玩法模式

### 5.1 排位赛（主模式）

标准 1v1，匹配段位相近的对手，赢加分输扣分。

### 5.2 ⚡ 闪电赛（休闲快节奏）

```
时长：1 个番茄钟（25min）
特点：下注翻倍，快速结算
适合：碎片时间、午休前
节奏：开战→专注25min→结算 全程不到30min
```

### 5.3 🏃 马拉松赛（深度专注）

```
时长：3~6 小时
特点：专注倍率上限提升至 ×5
      每完成 1 小时额外奖励
      中断惩罚翻倍
适合：周末大块时间、备考冲刺
```

### 5.4 👻 影子挑战（单人）

```
玩法：挑战"过去的自己"
      系统用用户上周同时间的专注数据生成"影子"
      和影子比赛，赢了证明自己在进步

特点：无需好友在线，随时可玩
      专注于自我提升而非击败他人
      适合新手熟悉系统
```

### 5.5 🏰 BOSS 战（双人合作）

```
玩法：两位好友组队，共同对抗 BOSS
      BOSS = 一个共同的目标专注时长
      两人专注时长之和达到目标 → BOSS 击败
      
      BOSS 设计：
      🐉 恶龙（2h）     → 两人共专注 2h
      🦑 海妖（5h）     → 两人共专注 5h
      👾 外星人（10h）  → 两人共专注 10h

特点：合作而非竞争，适合好友一起卷
      战胜 BOSS 获得限定合作徽章
      让不习惯竞争的用户也能参与
```

## 六、平衡性设计

### 6.1 反作弊

| 问题 | 解决方案 |
|------|---------|
| 挂机刷时长 | 需要定期交互（随机弹出"还在吗？"点击确认） |
| 多设备同时登录 | 同一账号只能一个设备在线 |
| 虚假专注 | 加速度传感器 + 屏幕触摸检测 |
| 小号刷分 | 新账号前 10 场不计入排位 |

### 6.2 防沉迷

| 机制 | 说明 |
|------|------|
| 每日对战上限 | 免费用户 10 场/天，Pass 用户 20 场/天 |
| 强制休息 | 连续专注 2h 后强制 15min 休息 |
| 深夜模式 | 0:00-6:00 仅开放单人影子挑战，不匹配排位 |
| 健康提醒 | 每完成一个番茄钟提示喝水/拉伸 |

### 6.3 公平竞技

| 原则 | 实现 |
|------|------|
| 不卖数值 | 商城只卖外观，不卖属性/加成 |
| 实力匹配 | ELO 匹配，同段位优先 |
| 新手保护 | 前 10 场仅匹配新手，不计入排位 |
| 保底机制 | 连胜终结时匹配 AI 对手保体验 |
