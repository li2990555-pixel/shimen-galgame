# 创意方案库：人文松（Humanities Hackathon）

## 一、基础信息

- **活动名称**: 人文松（Humanities Hackathon）
- **定位**: 无代码黑客松——保留黑客松"限时组队创作+有效产出"内核，将产出从代码替换为人文社科领域的具体内容作品
- **发起人背景**: 中文系硕士，无策展/技术/美术经验
- **首场规模**: 10-20人（3-4队，每队3-4人）
- **生成时间**: 2026-06-20

---

## 二、核心定位

### 人文松的"不变"——继承黑客松的基因

| 黑客松要素 | 人文松版本 |
|-----------|-----------|
| 限时完成挑战 | 1.5天限时创作——继承"极限感" |
| 跨学科组队 | 鼓励人文各科（中文/历史/哲学/社会学/艺术理论）混搭 |
| 有效产出 | 深度图文作品——可发表、可传播、真正"有效" |

### 人文松的"变"——不同于黑客松之处

| 维度 | 技术黑客松 | 人文松 |
|------|-----------|--------|
| 产出 | 可运行的app/网站 | 深度图文作品（特写/纪实/专题） |
| 核心能力 | 编程/设计 | 叙事/观察/分析/表达 |
| 工具 | IDE/框架/API | 纸笔/手机/协作文档/免费图库 |
| 门槛 | 需要技术基础 | 零门槛，人人可参与 |

### 人文松不同于什么

| 不是 | 是 |
|-----|----|
| 写作工作坊（无产出压力） | 限时产出，必有成果 |
| 学术会议（论文评议） | 面向公众的深度内容创作 |
| 艺术展/策展（需要美术功底） | 以文字为核心，图片为辅助 |
| 沙龙/读书会（漫谈） | 目标明确，成果导向 |

## 三、完整活动方案

### 活动框架

| 要素 | 内容 |
|------|------|
| **产出形式** | 深度图文作品（特写/纪实/专题文章，图文并茂，2000-5000字） |
| **活动时长** | 周六下午 13:00 → 周日下午 17:00（1.5天） |
| **参与者规模** | 首场10-20人，每队3-4人 |
| **主题机制** | 半开放命题——主办方给定大方向，各队自选切入角度 |
| **组队方式** | 兴趣驱动——先各自思考选题切口，再根据方向自然聚合 |
| **呈现形式** | 协作文档在线展示 + 精选作品后续公开发表 |
| **路演评委** | 邀请嘉宾评委（记者/学者/作家）参与点评 |

### 活动时间表

#### Day 1（周六）

| 时间 | 环节 | 说明 |
|------|------|------|
| 13:00-13:30 | 签到 & 破冰 | 每人分享一个最近让自己"停下来想了一下"的日常发现 |
| 13:30-14:00 | 开幕宣讲 | 公布大主题，解读规则和产出要求 |
| 14:00-15:00 | 个人思考时间 | 各自围绕主题写一段"我想探索的切口"（200字以内） |
| 15:00-16:00 | 组队环节 | 每人分享自己的切口，兴趣相近的人自然聚合，形成3-4队 |
| 16:00-18:00 | 团队讨论 | 确定最终选题、切入角度、大致结构、分工 |

#### Day 2（周日）

| 时间 | 环节 | 说明 |
|------|------|------|
| 9:00-9:30 | 晨间集合 | 各队快速同步进展，是否有卡点需要帮助 |
| 9:30-12:30 | 集中创作 | 采访/查资料/写作/配图——各队按分工推进 |
| 12:30-13:30 | 午休 | 自由交叉交流时间 |
| 13:30-14:30 | 冲刺收尾 | 最后修改、排版、校对 |
| 14:30-16:00 | **路演环节** | 每队6分钟展示成果 + 4分钟嘉宾/观众提问 |
| 16:00-17:00 | 自由交流 | 评选最佳作品、合影、结束 |

## 四、实操指南

### 筹备清单（首场）

- [ ] **找到2-3位联合发起人**——从同级同学中找（新闻系、社会学系、哲学系），分担组织工作
- [ ] **确定场地**——学校教室/图书馆研讨室/咖啡馆包场
- [ ] **确定大主题**——选择一个有张力的开放命题
- [ ] **招募参与者**——详见下方文案方向
- [ ] **邀请1-2位嘉宾评委**——记者、学者、作家皆可
- [ ] **准备工具清单**——告知参与者"带上手机和想法即可"

### 招募文案核心卖点

| 卖点 | 文案方向 |
|------|---------|
| 文科生的黑客松 | "写论文的人也需要一次极限创作体验" |
| 低门槛 | "不需要技术，不需要美术，只需要你对世界有好奇心" |
| 有效产出 | "一个周末，带走一篇可以发表的作品" |
| 跨学科碰撞 | "和学历史/哲学/社会学的人做一件作品" |

### 推荐首场主题方向

1. **"附近"** ——重新发现身边被忽略的人和事
2. **"消失中"** ——记录正在消失的语言/空间/习俗/记忆
3. **"陌生人"** ——城市的匿名性和人际可能
4. **"日常的仪式感"** ——那些我们每天做但没想过的行为
5. **"边界"** ——代际/阶层/城乡/文化的交汇地带
