- **生成时间**: 2026-06-29

## 二、市场竞品扫描

| 竞品 | 形态 | 特点 | 差距机会 |
|------|------|------|---------|
| 天机（来事儿AI） | 开源大模型 | 七大领域，技术路线含 Prompt/Agent/知识库 | 泛化通用，无地域特色 |

## 三、核心创意方案

### 方案 A：「山东通」— 山东人情世故全能助手（推荐）

**核心思路**: 以"山东人情世故"为文化底座，打造一个懂规矩、会说话、能策略性处事的AI助手。不只是话术模板，而是"文化逻辑 + 场景策略 + 智能生成"三位一体。

**三层能力架构**:

```
第一层：文化根基
  ├── 山东人情文化底层逻辑
  │   ├── 儒家传统：尊卑有序、长幼有别、礼尚往来
  │   ├── 官本位思想：酒桌即小官场，规矩与身份地位挂钩
  │   ├── 人情社会逻辑：互惠原则 + 面子维护 + 关系储备
  │   └── 豪爽直率：实在、义气、不扭捏
  └── 场景知识库
      ├── 酒桌座次规则（主陪/副陪/主宾/副宾位置逻辑）
      ├── 敬酒顺序与话术（带酒/打圈/鱼头酒）
      ├── 送礼潜规则（时机/品类/话术/避讳）
      ├── 办事开口艺术（铺垫/时机/被拒应对）
      └── 人情往来的"账本"逻辑（谁欠谁、何时还）

第二层：场景策略引擎
  ├── 场景识别 → 判断这是什么局（饭局/送礼/求人/拒绝）
  ├── 角色定位 → 你是谁（求人者/被求者/陪客/主人/客人）
  ├── 利益计算 → 这束光有没有利？无利就灭光
  └── 策略输出 → 先做什么、再说什么、底线在哪

第三层：话术生成器
  ├── 根据策略自动生成个性化话术
  ├── 风格可调（正式/幽默/谦卑/豪爽）
  ├── 支持多轮对话推演（如果对方这样说，你怎么接）
  └── 附带"避坑提醒"（这话术里哪些词可能踩雷）
```

**典型交互示例**:

```
用户：明天要和山东的导师吃饭，我是研一新生，第一次上酒桌，怎么办？

→ 【文化解读】山东酒桌重长幼有序，导师=主宾，你=末位陪客
→ 【座次指南】进门先让导师坐主位，你坐门口或上菜位
→ 【话术建议】
  - 开场：导师好，我是XX，以后请您多指导（双手举杯，杯沿低于对方）
  - 敬酒：导师，我敬您一杯，您随意（先干为敬）
  - 被夸：没有没有，还有很多要跟您学的
→ 【避坑】不要在导师发言前单独敬别人；不要抢主陪的风头
```

### 方案 B：「人情研究所」— 互动式场景推演

**核心思路**: 不是直接给答案，而是跟用户一起"演习"。用户描述一个真实困境，AI扮演场景中的其他角色，进行多轮对话模拟，事后复盘。

**核心功能**:
1. **角色扮演模式**: AI扮演领导/长辈/客户，用户练习应对，AI实时反馈
2. **复盘分析**: 对话结束后，AI分析用户哪些话得分、哪些失分
3. **多分支推演**: "如果你选A，对方会这样回应；如果你选B，则..."

**适用场景**: 社恐练习、面试预演、求人办事前的心理建设

**与方案A的关系**: 方案A做"防守型工具"（遇到问题了来查），方案B做"训练型工具"（提前练习）。建议方案A为底座，方案B为进阶。

---

### 方案 C：「人情账本」— 关系管理工具

**核心思路**: 山东人讲究"心里有本账"——谁帮过我、我欠谁人情、什么时候该还。把这个隐形账本做成显性工具。

**核心功能**:
1. **人情记录**: 记录谁帮了你什么、你帮了谁什么
2. **提醒系统**: "张老师三个月前帮你写了推荐信，该找个机会表示一下了"
3. **关系图谱**: 可视化你的社交网络，标注亲疏远近
4. **随礼参考**: 根据关系远近、场合类型、当地行情推荐随礼金额

**与方案A的关系**: 可作为方案A的外围功能或独立工具。偏向"管理型"，而非"生成型"。

## 五、MVP 建议

**第一版（Prompt 形态）核心能力**:

1. **核心场景 x 5**:
   - 酒桌礼仪（座次/敬酒/挡酒/话术）
   - 送礼策略（选品/时机/话术/避讳）
   - 求人办事（开口/铺垫/被拒应对）
   - 人情答复（如何体面拒绝不伤面子）
   - 师门相处（导师/同门/学术场合）

2. **三层输出格式**: 文化解读 → 策略建议 → 具体话术

3. **风格切换**: 豪爽派 / 谦卑派 / 幽默派

**后续迭代方向**:
   - v2: 加入角色扮演推演（方案B）
   - v3: 加入人情账本功能（方案C）
   - v4: 升级为 Claude Code Skill，加入知识库
