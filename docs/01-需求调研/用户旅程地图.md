# 用户旅程地图：人文松参与者体验

## 一、用户画像

- **用户类型**: 首次参与者
- **旅程范围**: 从发现活动到活动结束的全流程
- **背景**: 人文社科在校生/青年，对人文议题有兴趣，有表达欲但没有合适的表达场合
- **生成时间**: 2026-06-20

---

## 二、旅程概览

```
发现与报名 → 到场与破冰 → 主题与组队 → 创作冲刺 → 路演展示 → 结束与延伸
```

---

## 三、详细旅程分析

### 阶段 1: 发现与报名

**用户行为**:
- 在校园群/朋友圈里看到人文松的招募帖
- 滑到长图/文案，阅读活动介绍
- 纠结"我适不适合参加"、"会不会需要特殊技能"
- 最终扫码填写报名表
- 收到确认信息，等待活动日到来

**触点**: 微信群 → 招募帖（长图/飞书文档） → 报名表单 → 确认通知

**用户情绪**: 好奇 😊 → 犹豫 😐 → 期待 😊

**情绪值**: 3→4（上线波动）

**痛点**:
- "人文松到底是什么"——概念太新，第一眼可能看不懂（中等）
- "我没特殊技能，能参加吗"——对门槛的担忧（中等）

**机会点**:
- 招募文案用一句话说清"这是什么"+"不用代码/美术"+"带走一个作品"
- 放一个往期案例/模拟示例，让人看到"别人做了什么"
- 报名后自动回复一条"你适合参加吗"的轻松说明

---

### 阶段 2: 到场与破冰

**用户行为**:
- 周六下午到达场地，签到
- 看到场地环境，感受氛围
- 参加破冰环节，分享一个"最近的日常发现"
- 听其他人分享，初步认识在场的人

**触点**: 场地入口 → 签到处 → 座位/活动空间 → 破冰环节

**用户情绪**: 好奇兴奋 😊

**情绪值**: 4

**痛点**:
- 一个人来，不认识任何人——社交压力（轻微）
- 不知道别人是什么学科背景——对跨度的未知（轻微）

**机会点**:
- 签到时让每个人写一个"关键词+学科"贴身上，一眼可读
- 破冰设计成"低压力"的分享，不要求自我介绍式的"我是谁"

---

### 阶段 3: 主题与组队

**用户行为**:
- 听开幕宣讲公布半开放主题
- 个人独立思考时间——写一段"我想探索的切入点"
- 每人分享自己的切口
- 听别人的分享，寻找兴趣相近的人
- 自然聚合成3-4人的队伍
- 队内讨论：确定选题、角度、分工

**触点**: 开幕宣讲 → 个人思考 → 组队环节 → 团队讨论

**用户情绪**: 有点迷茫 😕 → 逐渐清晰 😊

**情绪值**: 3→4（先降后升）

**痛点**:
- "主题好大，我能从哪里切入"——不知道选什么切口（中等）
- 想做的方向没人一起——落单风险（中等）
- 组队后方向不确定——讨论效率低（轻微）

**机会点**:
- 事先准备3-5个"切口示例"抛砖引玉
- 预留充足的组队时间（至少30-45分钟）
- 每组配一张"选题工作纸"引导讨论结构

---

### 阶段 4: 创作冲刺

**用户行为**:
- 确定分工（谁主笔/谁找素材/谁做视觉等）
- 收集素材、采访/调研/拍摄
- 实际创作——写作/拍摄/录制/设计/搭建
- 遇到卡点时团队讨论突破
- 通宵或早起赶工
- 冲刺收尾：修改、校对、排版

**触点**: 创作空间 → 团队讨论 → 必要时外出采集 → 冲刺阶段

**用户情绪**: 冲劲满满 😊 → 卡住焦虑 😕 → 冲刺亢奋 😄

**情绪值**: 5→2→5（经典黑客松曲线）

**痛点**:
- 创作到一半发现选题太大，做不完（严重）
- 团队成员节奏不同步（中等）
- 工具/素材不够（轻微）
- 通宵后第二天状态差（中等）

**机会点**:
- 开幕时告知"选题宁小勿大"——一个小切口深入胜过全景扫描
- 设一个"中途检查点"（周六晚上），各队快速同步，主办方检查是否需要缩小范围
- 准备工具清单提前发给参与者
- 提供零食/咖啡等基础补给

---

### 阶段 5: 路演展示

**用户行为**:
- 准备展示内容（口头讲稿/展示材料/演示准备）
- 按顺序上台展示成果（6分钟展示+4分钟问答）
- 观看其他队伍的展示
- 接受嘉宾/同伴的反馈和提问

**触点**: 路演舞台/投影区 → 观众席 → 问答环节

**用户情绪**: 紧张 😅 → 自豪 😊

**情绪值**: 3→5（先紧后松）

**痛点**:
- 时间不够，展示时只能呈现部分内容（中等）
- 临场紧张，不知道怎么讲清楚（轻微）
- 成果未达到预期，怕被对比（轻微）

**机会点**:
- 提前给一个"展示框架"：我们做了什么→为什么选这个→最大的发现/收获是什么
- 强调"展示是为了交流而不是比赛"
- 嘉宾反馈以"发现亮点"为主，而非批判

---

### 阶段 6: 结束与延伸

**用户行为**:
- 自由交流，和其他队伍交换想法
- 合影留念
- 离开活动场地
- 后续在群里分享作品/照片
- 看到自己的作品被发到公众号/朋友圈

**触点**: 活动空间 → 微信群 → 公众号/社交媒体

**用户情绪**: 意犹未尽 😊

**情绪值**: 4

**痛点**:
- 活动结束就断了，缺少后续连接（中等）
- 作品停留在活动当天，没有进一步发表/展示的机会（中等）

**机会点**:
- 建立参与者社群，持续互动
- 精选作品后续发表在公众号/合作平台
- 设计"人文松校友"身份，下一场可以当导师/评委
- 活动后一周内发回顾推文，保持余温

## 七、改进建议

### 核心优化（让体验从及格到优秀）

1. **P0：招募文案解决"看不懂+怕不配"**
   - 结构：一句话定义 → 谁适合参加 → 你会带走什么 → 具体怎么做 → 报名
   - 放一个"模拟示例"让人看到别人会产出什么

2. **P0：选题引导解决"大主题下无从下手"**
   - 公布主题时同步给出3-5个切口方向示例
   - 个人思考阶段给一张"选题工作纸"（引导填空式）

3. **P0：中途检查点防止创作崩盘**
   - 周六晚上设15分钟的快速同步，每个队30秒说"我们在做什么"
   - 主办方判断是否选题过大，及时帮团队缩小范围

4. **P1：路演模板降低展示焦虑**
   - 给一个简单框架：我们做了什么→为什么选这个→最大的发现/收获

5. **P1：活动后持续连接**
   - 建群+一周内发回顾+精选作品发表
