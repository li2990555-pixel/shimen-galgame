# 师门galgame攻略助手

帮你解码师门人情场景中的潜台词，提供体面的应对策略——就像在玩一款真实版师门 Galgame，每条对话都影响好感度。

> 你不必学会讨好，但可以不踩坑。

---

## 这是什么？

**师门galgame攻略助手** 是一个 Claude Code Skill / Prompt，专注于解码师门人情场景中的潜台词，并提供具体的应对策略。

它基于真实调研（知乎、豆瓣、虎扑等平台上高校师门文化、人情往来的真实案例），不是凭空编造的话术库。

### 四种工作模式

| 模式 | 作用 | 用法 | 输出 |
|------|------|------|------|
| **场景分析** | 单次话术解码 + 应对策略 | 描述一个场景 | 潜台词分析 + 话术建议 |
| **人物蒸馏** | 跨对话摸透一个人的性格和套路 | 收集同一个人的多段话 | 社交肖像 + 操作手册 |
| **关系诊断** | 判断你和某人的关系状态 | 列出近期几次互动 | **好感度面板 + 维度雷达** |
| **关系网络** | 以你为中心的全局关系视图 | 列出已有诊断结果 | **星状网络图 + 全景快照 + 边缘预警** |

四种模式可以串联使用：**先摸清一个人 → 再判断关系状态 → 再拼出全局网络 → 最后给出具体应对**。

### 关系诊断的好感度面板

首次诊断某个人时，会先问一句"你们之前整体关系怎么样？1-10打分"来确定好感度基线，再根据每条互动加减分数。

### 核心定位

- **用户**：反感"重礼开路"方式、但不想因不懂规矩吃亏的年轻人（硕博生、职场新人）
- **价值观**：不教人更会讨好，而是教人在不想做的时候能体面地不做的能力
- **差异化**：识局能力（对方潜台词解码）+ 策略建议（做不做、怎么做、底线在哪），而不是背诵话术模板

### 覆盖场景

- 师门相处（含学术饭局行为指南、导师类型判断）
- 课题组竞争（同门合作与边界、数据保护）
- 谢师宴/毕业请客压力应对
- 人情答复（体面拒绝的策略和话术）

> 所有场景知识库内容基于真实调研（知乎、豆瓣、虎扑、澎湃新闻、科学网等平台上硕博生的真实案例），不是凭空编造的话术库。

## 快速开始

### 方式一：作为 Claude Code Skill 使用

确保 `shimen-galgame` skill 已安装：

```bash
ls ~/.claude/skills/shimen-galgame/SKILL.md
```

然后在对话中直接输入场景即可：

```
导师让我帮他拿快递取烟酒，研一新生，去还是不去？

帮我分析一下我导师这个人——他跟我说过四句话：
"年轻人多锻炼锻炼"、"这个方向很有前景"、"再改改"、"XX是我最看好的"

帮我看看我跟导师最近的关系：
1. 上周他让我写申请书，我没接话
2. 这周组会他跳过了我的汇报
3. 今天他主动约我聊
```

### 方式二：直接作为 Prompt 使用

`skill/SKILL.md` 是完整的 Prompt 文件，可直接复制到支持 System Prompt 的 Claude 产品中使用。

## 设计原则

1. **不教人使诈** — 只教"懂规矩、会来事、不踩雷"
2. **帮用户看懂局面，再自己做决定**
3. **所有建议以"不惹事、不越界、保护自己"为前提"
