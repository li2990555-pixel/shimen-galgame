# 鲁门心法

帮"夹缝中的年轻人"解码人情场景中的潜台词，提供体面的应对策略。

> 你不必学会讨好，但可以不踩坑。

---

## 这是什么？

**鲁门心法** 是一个 Claude Code Skill / Prompt，专注于解码中国式人情世故场景中的潜台词，并提供具体的应对策略。

它基于真实调研（知乎、豆瓣、虎扑等平台上山东高校师门文化、人情往来的真实案例），不是凭空编造的话术库。

### 三种工作模式

| 模式 | 作用 | 用法 |
|------|------|------|
| **场景分析** | 单次话术解码 + 应对策略 | 描述一个场景即可 |
| **人物蒸馏** | 跨对话摸透一个人的性格和套路 | 收集同一个人的多段话 |
| **关系诊断** | 判断你和某人的关系状态 | 列出近期几次互动 |

三种模式可以串联使用：**先摸清一个人 → 再判断关系状态 → 最后给出具体应对**。

### 核心定位

- **用户**：反感"重礼开路"方式、但不想因不懂规矩吃亏的年轻人（硕博生、职场新人）
- **价值观**：不教人更会讨好，而是教人在不想做的时候能体面地不做的能力
- **差异化**：识局能力（对方潜台词解码）+ 策略建议（做不做、怎么做、底线在哪），而不是背诵话术模板

### MVP 覆盖场景

- 师门相处（含学术饭局行为指南）
- 人情答复（体面拒绝的策略和话术）

## 快速开始

### 方式一：作为 Claude Code Skill 使用

确保 `lumian-xinfa` skill 已安装：

```bash
ls ~/.claude/skills/lumian-xinfa/SKILL.md
```

然后在对话中直接输入场景即可：

```
导师让我帮他拿快递取烟酒，研一新生，去还是不去？

山东老乡找我借钱，不太熟，怎么拒绝不伤面子？
```

### 方式二：直接作为 Prompt 使用

`skill/SKILL.md` 是完整的 Prompt 文件，可直接复制到支持 System Prompt 的 Claude 产品中使用。

## 设计原则

1. **不模仿方言** — 使用自然普通话，不刻意加入倒装句或语气词
2. **不教人使诈** — 只教"懂规矩、会来事、不踩雷"
3. **帮用户看懂局面，再自己做决定**
4. **所有建议以"不惹事、不越界、保护自己"为前提"
